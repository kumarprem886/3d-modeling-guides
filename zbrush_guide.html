ZBrush Guide
Digital Sculpting
Important: ZBrush uses a very unique UI — it is NOT a standard 3D application. The canvas is a 2.5D paint surface; your model exists on it. Take time to learn the canvas vs 3D model distinction before diving into shortcuts — many operations behave differently depending on whether you are in Edit mode or not.

Navigation

ZBrush has NO traditional orbit control. Navigation works through modifier key combinations on the canvas — not a middle mouse button or numpad like most 3D apps.

Core Navigation Controls

ActionShortcutNotes
Orbit / Rotate Alt + Drag (on model) Drag on the model while holding Alt to rotate the view around it
Frame / Reset view Alt + Click (empty area) Click on an empty area of the canvas to snap back to a neutral view
Zoom (canvas) Ctrl + Drag Zooms the canvas — distinct from zooming the 3D model itself
Pan / Move Shift + Drag (right-click drag) Pans the canvas left/right/up/down
Frame model F Fits the active subtool into the viewport
Perspective toggle P Switches between perspective and orthographic projection
Floor grid Shift + P Toggles the floor grid display
Transform widget (Gizmo 3D) Y Toggles the Gizmo 3D transform widget on/off

Canvas vs Model Zoom

CCanvas Zoom

Uses Ctrl + drag. This zooms the entire canvas view — like zooming a photo. Your model gets bigger on screen but its actual 3D size has not changed. Use this for viewport navigation.

MModel Zoom

Use the Scale gizmo (E) or Transform > Scale to resize the actual 3D object. The Zoom slider in the right tray also adjusts the 3D zoom level independently.

Tip: If your model disappears or the canvas looks blank, press F to re-frame, or Alt + click on empty canvas to reset the view position.

Stroke Types & Top Bar Modes

SStroke Types

Spray — Scatters multiple brush stamps along the stroke path. Good for textures and organic variation.

DragRect — Draws one brush stamp by dragging a rectangle. Used for precise alpha placement.

MTop Bar Mode Buttons

Mrgb — Affects both Material and RGB color when sculpting/painting

Rgb — Affects color only, not depth

M — Affects material only, not color or depth

Polypainting & Texturing

Polypainting stores color directly on mesh vertices — no UVs required for painting. UVs are only needed when you bake textures for export.

Enable & Paint

StepActionNotes
1. Enable Tool → Polypaint → Colorize (toggle on) Activates vertex color storage on the mesh
2. Set mode Rgb or Mrgb on top bar Rgb = color only, Mrgb = color + material
3. Set intensity I → drag for RGB intensity Lower intensity for subtle color blending
4. Paint Use Standard or Color spray brush on the surface Z Intensity should be 0 if you only want to paint color
5. Fill object Tool → Polypaint → Fill Object Floods the entire mesh with the active color

Intensity Controls

RRGB Intensity

Controls how strongly color is applied per brush stroke. Press I and drag. Low values (5–20) allow gradual color blending like real paint.

ZZ Intensity

Controls sculpt depth. When polypainting, set Z Intensity to 0 so strokes do not push/pull the surface — only apply color.

UV Unwrap & Texture Baking

TaskTool / Location
UV unwrap ZPlugin → UV Master → Unwrap. Enable Polygroups option for seam control.
Check UVs Tool → UV Map → Morph UV to flatten and inspect UV layout
Bake all maps ZPlugin → Multi Map Exporter → select map types → Export All Maps
Map types available Color / Diffuse, Normal, Displacement, Ambient Occlusion, Cavity
Texture resolution Set in Multi Map Exporter — common values: 2048, 4096, 8192
Workflow note: Always run UV Master before exporting textures. ZBrush cannot bake texture maps without valid UVs on the mesh.

Concept to Export Workflow

A standard ZBrush production workflow — from first DynaMesh block-out through to exporting maps for a game engine or rendering application.

  1. 1
    Block Out Primary Forms with DynaMesh Start from a DynaMesh sphere (Tool → DynaMesh, enable). Sculpt large shapes using ClayBuildup and Move brushes. Do not worry about topology yet. Remesh frequently with Ctrl + drag on empty canvas to keep polygon distribution even. Adjust resolution as forms get more complex.
  2. 2
    Retopologize with ZRemesher Once primary forms are solid, run Tool → Geometry → ZRemesher to generate clean quad topology. Set target polygon count appropriate for your level of detail (e.g. 5K–20K for a character base). Use ZRemesherGuide brush to hint edge flow around eyes, mouth, and limbs if needed.
  3. 3
    Subdivide and Sculpt Secondary & Tertiary Detail Subdivide with Ctrl + D up to the level needed (typically 4–6 for detailed character work). Navigate levels with D and Shift + D. Add secondary forms, muscle structure, skin folds. Use Dam_Standard for creases and Slash3 for fine surface lines. Use TrimDynamic and hPolish for clean planar surfaces.
  4. 4
    Polypaint Surface Color & Details Enable Tool → Polypaint → Colorize. Switch the top bar to Rgb mode and set Z Intensity to 0. Paint color gradients, skin tones, or material zones using the Standard brush at low RGB intensity. Use Fill Object for base color floods. Use Spray stroke for texture variation.
  5. 5
    UV Unwrap with UV Master Run ZPlugin → UV Master → Unwrap to generate automatic UVs. Enable Polygroups to use your existing polygroup seams. Check the result with Tool → UV Map → Morph UV. Re-unwrap if needed with Protect and Attract painting to guide seam placement.
  6. 6
    Bake Maps with Multi Map Exporter Open ZPlugin → Multi Map Exporter. Select Color, Normal, Displacement, and AO maps. Set resolution (4096 recommended). Click Export All Maps. Normal and displacement maps capture the high-poly detail and apply it to the low-poly base mesh in your target application.
  7. 7
    Export Base Mesh as OBJ or via GoZ Go to the lowest subdivision level (Shift + D repeatedly). Export as OBJ: Tool → Export → OBJ. Alternatively, use Tool → GoZ to send directly to Blender, Maya, or Cinema 4D. Import the OBJ and baked maps into your renderer or game engine, apply the normal/displacement maps to recover detail.

Quick Pipeline Summary

1Blocking

DynaMesh sphere → ClayBuildup + Move → Ctrl+drag to remesh

2Retopo

ZRemesher → clean quads → ready for subdivision

3Detail

Ctrl+D subdivide → Dam_Standard, TrimDynamic, Slash3

4Paint

Polypaint → UV Master → Multi Map Exporter

5Export

OBJ (low subdiv) + baked maps → Blender / game engine

Remember: ZBrush is a non-destructive, multi-resolution sculpting app. You can always go back to a lower subdivision level to adjust forms without losing high-poly detail. Plan your subdivision workflow before starting detail work.
