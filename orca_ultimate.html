🦭Understanding OrcaSlicer

What it is, why it matters, and how to think about slicing Beginner

What is OrcaSlicer?

OrcaSlicer is a free, open-source slicer for FDM and resin 3D printers. It was forked from Bambu Studio (which itself forked from PrusaSlicer), combining the best of both with its own powerful additions — especially its built-in calibration suite.

Bottom line: OrcaSlicer is widely considered the best free slicer for advanced users who want granular control, excellent calibration tools, and compatibility with virtually any printer.
Why Choose OrcaSlicer?
  • Built-in calibration tools — pressure advance, flow rate, temperature tower, retraction, VFA tests all in one menu
  • Advanced multi-material support with purge tower control
  • Pressure Advance / Linear Advance tuning per-filament
  • Flow calibration wizard for every new spool
  • Better bridging algorithms than Cura in most scenarios
  • Arachne wall generator for variable-width perimeters
  • Active development — frequent releases, large community
Supported Printer Brands
  • Bambu Lab — native profiles (X1, P1, A1 series)
  • Prusa — MK3S+, MK4, Mini, XL
  • Creality — Ender 3/5, CR-10, K1, Ender-6
  • Voron — 0, 2.4, Trident, Switchwire
  • Ratrig — V-Core, V-Minion
  • Any custom printer — configure from scratch via printer wizard
  • Resin printers (MSLA/DLP) — basic support
If your printer isn't listed, import a Klipper config or start from a generic profile and customize it.

Key Slicing Concepts

Core Settings

  • Layer Height — vertical resolution of each printed layer. Smaller = finer detail, slower print.
  • Infill — internal structure of the part. Pattern and density control strength and material use.
  • Supports — temporary structures for overhangs that cannot print in mid-air.
  • Brim / Raft — extra material around/under the first layer to improve bed adhesion.
  • Perimeters / Walls — outer shell loops that define the part's exterior shape.
  • Top/Bottom Layers — solid layers that close the shell above and below infill.

The Print Quality Triangle

Speed vs Quality vs Strength Increasing one always trades off another. You cannot maximise all three simultaneously. Most prints live in the middle — the skill is knowing which to sacrifice for your use case.
  • Speed up → reduce quality, may reduce layer bonding
  • Slow down → better quality, longer print times
  • More walls/infill → stronger, heavier, uses more filament

Calibration Philosophy

OrcaSlicer's biggest competitive advantage is its Calibration menu. Unlike Cura or PrusaSlicer, it includes guided tests for every major variable. The philosophy is: tune the machine, then tune the filament, then print.

Run calibrations every time you switch filament brand or colour. Even same-brand different-colour spools can have measurably different flow rates.
5 Things to Do First
1
Install your printer profile — Printer menu > Add Printer. Use an official profile if available.
2
Set your filament brand and type — use a matching preset as the starting point, then customise temperatures.
3
Run Flow Rate Calibration — Calibration menu > Flow Rate. Measure and enter the value.
4
Run Pressure Advance (PA) Calibration — find the value where corners are sharpest and cleanest.
5
Do a Temperature Tower — find the ideal print temperature for your specific spool.

Layer height, walls, top/bottom surfaces, ironing, and quality tweaks Intermediate

Layer Height
HeightUse CaseNotes

0.15–0.2mmStandard / general purposeBest balance of quality and speed. 0.2mm is the universal default.

First Layer Height: Set to 0.2mm regardless of layer height. A slightly taller first layer improves adhesion on any nozzle size.

Variable Layer Height

Right-click a model → Variable Layer Height. Drag the curve to reduce height on curved/detailed areas and increase on flat vertical sections. Reduces print time while keeping detail where it counts.

Adaptive Layer Height

Automatically adjusts layer height based on surface angle — thinner layers on shallow angles (smooth curves), thicker on steep vertical walls. Enable in Quality tab. Set Quality factor (lower = more adaptive, more layers).

Walls / Perimeters

Wall Count Guide

  • 2 walls — weak shell, decorative models only
  • 3 walls — standard, good for most prints
  • 4–5 walls — strong functional parts, recommended for mechanical use
  • 6+ walls — maximum strength, approaching solid shell
Wall count increases strength more efficiently than infill density for most load cases.

Wall Generator

  • Classic — fixed-width concentric loops. Simple, predictable.
  • Arachne (recommended) — variable-width walls that adapt to narrow features. Fills gaps that classic mode would leave unfilled. Better surface quality on organic shapes.

Detect Thin Walls

Enable to print features thinner than one nozzle width. Slows slicing slightly. Essential for fine models.

Seam Placement

OptionBehaviourBest For
NearestShortest travel to seam — random position each layerFast prints, seam visibility is acceptable
AlignedAll seams stack on the same XY position — creates a vertical lineWhen you want a predictable seam you can sand
RearForces seam to back of printVisual models seen from front only
ContourFollows object outline — seam follows sharp corners/edgesModels with natural edge lines to hide seam in

Wall Speeds

  • Outer wall: keep slow — 30–60 mm/s for best surface quality. This is the visible face.
  • Inner walls: can be 2–3× faster than outer — quality doesn't affect appearance directly.
Top & Bottom Surfaces

Shell Layer Counts

  • Top shell layers: 4–6 — fewer layers risks infill showing through the top surface. Increase if you see a bumpy/open top.
  • Bottom shell layers: 3–4 — less stress from below; can be slightly fewer than top.

Top/Bottom Patterns

  • Monotonic (recommended) — all lines same direction, fewer visible seams
  • Hilbert Curve — space-filling curve, very strong, shorter travel moves
  • Concentric — loops inward from perimeter, flexible appearance, good for flexible filaments

Ironing

After printing the top layer normally, ironing makes an extra slow pass at low flow rate to flatten surface imperfections — producing a glass-smooth finish.

  • Enable: Quality tab > Ironing
  • Ironing type: All top surfaces or Topmost surface only
  • Speed: 15–25 mm/s
  • Flow rate: 10–15%
Note: Ironing adds significant print time on large flat surfaces. Use selectively on display/visible faces.

Structural Strength Tips

Walls vs Infill for Strength

Key insight: 4 walls + 20% infill is almost always stronger than 2 walls + 50% infill for impact and bending loads. Walls carry load; infill provides internal support for top/bottom surfaces.
  • Add walls before increasing infill for structural applications
  • 4 walls at any reasonable infill density is the sweet spot for functional parts

Print Orientation

  • Inter-layer Z bonds are the weakest point in any FDM print
  • Orient so the primary load direction is in the XY plane, not Z
  • A vertical bar printed upright will snap at layer lines; printed lying flat it resists the same load far better

Interface & Overlap Settings

  • Interface layers — solid layers between infill and walls improve top surface quality
  • Infill/wall overlap: 25–30% — ensures infill bonds to walls; too low = gaps, too high = over-extrusion at junctions
  • Infill anchor length — how far infill extends into walls — longer = better bonding

Top Surface Strength

  • If top surface shows infill pattern through it (pillowing), increase top shell layers or lower layer height
  • Top solid layers require at least 4 layers at 0.2mm to fully close over standard infill
Modifier Meshes — Zone-Based Settings

Modifier meshes let you apply different settings to specific regions of your model without changing global settings.

How to Use

1
Right-click object in viewport → Add Modifier → choose a primitive (box, sphere, cylinder) or import an STL
2
Position and resize the modifier mesh to cover the region you want to modify (e.g., around a screw hole)
3
In the modifier's settings, override: infill density, wall count, top/bottom layers as needed
4
Slice — the region inside the modifier uses the override, rest of model uses global settings

Common Use Cases

  • Screw holes / inserts — high infill (60–80%) around fastener zone, 15% elsewhere
  • Stress concentration zones — extra walls in a narrow bridge or cantilevered section
  • Display face — ironing only on the visible top face without ironing the entire model
  • Weight reduction — low infill in a large bulk section, full density at load-bearing edges
Modifier meshes are one of OrcaSlicer's most powerful features for optimising functional prints. Learn them — they eliminate the need to split models in CAD.

Support Painting

Paint supports directly onto the model surface instead of relying on auto-generation everywhere.

  • Blue brush — enforce support here (adds support where the auto algorithm wouldn't place it)
  • Red brush — block support here (removes auto-generated support from that area)
  • Brush size — adjust with scroll wheel or slider for precision
Use case: Auto-generates supports on a decorative face you want clean — paint red to block them. A critical overhang isn't getting support — paint blue to force it.
Support Enforcers & Blockers

Primitive shape approach (vs. paint approach) for support control:

  • Right-click model in viewport → Add support enforcer or Add support blocker
  • Choose shape: cube, sphere, cylinder
  • Resize and position over the area to control
  • Enforcers appear green, blockers appear red as translucent overlays
Blockers are faster than paint for large flat areas. Paint is better for irregular contoured surfaces.
Bridging — Printing Horizontal Spans Without Support

A bridge is a horizontal span between two supported points. With a well-tuned printer, bridges can span significant distances without support.

Bridge Capabilities

  • Typical tuned printer: 50–60mm clean bridge
  • Well-tuned with good cooling: up to 80–100mm
  • Test with a bridge torture test file before relying on it

Bridge Settings

  • Bridge speed: slow down to 20–30 mm/s — gives filament time to solidify mid-air
  • Bridge flow: reduce to 0.90–0.95 — slight under-extrusion reduces sag

Cooling for Bridges

  • Set fan speed during bridging to 100% — maximum airflow solidifies the filament before it sags
  • Bridging direction: OrcaSlicer automatically orients bridge lines in the optimal direction (perpendicular to the span)
PLA bridges well. PETG sags more. ABS/ASA bridge poorly without enclosure. TPU barely bridges. Adjust bridge speed and flow per material.
Soluble Supports (Multi-Material)
  • PVA — dissolves in water. Works with PLA. Brittle, humidity-sensitive, expensive.
  • BVOH — dissolves in water. Easier to print than PVA, better layer adhesion, less brittle.
  • HIPS — dissolves in limonene. Pair with ABS for enclosed printer setups.
In OrcaSlicer multi-material settings, assign the support material to Extruder 2 and enable Soluble support interface — only the interface layers use the expensive soluble filament, bulk support uses cheap PLA.

🧪Calibration — OrcaSlicer's Superpower

Every built-in calibration test explained with target values and interpretation Advanced

Access all calibration tests: Top menu bar > Calibration. Run these in order on every new filament: Temperature Tower → Flow Rate → Pressure Advance → Retraction. Then set Max Volumetric Speed for high-speed printing.
1. Temperature Tower

Prints a tower that automatically changes printing temperature at set height intervals. Each band represents a different temperature. Find the band with the best combination of: no stringing, clean bridging, and good layer adhesion.

Typical Ranges by Material

MaterialRangeStart Temp
PLA190–230°C220°C, −5°C/section
PETG220–250°C245°C, −5°C/section
ABS220–250°C245°C, −5°C/section
TPU210–240°C230°C, −5°C/section

What to Look For

  • Too hot: stringing between towers, blobbing, oozing at corners, drooping bridges
  • Too cold: poor layer bonding, brittle result, rough surface texture, under-extrusion, white stress lines
  • Just right: clean bridges, minimal stringing, smooth surface, sharp corners, good layer fusion
Save the winning temperature in your filament profile. Repeat for each new spool — even same brand/colour batches can vary by ±5°C.
2. Flow Rate Calibration

Ensures your printer extrudes exactly the right amount of plastic. Incorrect flow is the root cause of most surface quality issues — over-extrusion causes bumpy tops; under-extrusion causes gaps and weak layers.

Method: OrcaSlicer Two-Step Flow Calibration

1
Volumetric Flow (Step 1): Print the single-wall cube from the calibration menu. Measure the wall thickness with calipers. Compare to expected (nozzle diameter). Calculate: new_flow = (target_width / measured_width) × current_flow_ratio
2
Fine Flow (Step 2): OrcaSlicer prints a set of reference cubes with incrementally varying flow. Inspect visually — find the cube with the flattest, most uniform top surface. Enter that value in the dialog.
3
Save the new flow ratio in your filament profile. Label the profile with brand, colour, and date.
Do this for every new filament spool and brand. Flow ratios typically range from 0.92 to 1.05 — cheap filaments often need more correction than quality brands.
3. Pressure Advance (PA) / Linear Advance

Pressure Advance compensates for the pressure buildup in the nozzle during acceleration and deceleration. Without it, corners have blobs (too much pressure at slow speeds) or rounding (not enough pressure to fill corners at high speed).

Typical PA Values

MaterialDirect DriveBowden
PLA0.02–0.060.4–0.8
PETG0.05–0.150.8–1.5
ABS0.03–0.080.5–1.0
TPU0.1–0.5N/A (not recommended)

OrcaSlicer PA Tower Method

  • Calibration menu > Pressure Advance
  • Prints a tall tower with varying PA value each section (shown on side)
  • Find the section where corners are sharpest and cleanest — no blobs, no rounding
  • Enter that value in your filament profile under Pressure Advance
Direct drive printers need much lower PA values than Bowden setups — the shorter filament path means less pressure builds up.
4. Retraction Calibration

Retraction pulls filament back when the nozzle travels between points — preventing strings/blobs of ooze. Too little = stringing. Too much = gaps, grinding of filament, clogs.

Target Values

Extruder TypeDistanceSpeed
Direct Drive0.5–1.5mm25–45 mm/s
Bowden (PTFE)3–7mm40–60 mm/s

Calibration Method

Calibration menu > Retraction. Prints a stringing test tower with increasing retraction distance each section. Find the lowest distance that eliminates stringing — using the minimum necessary reduces wear on filament and prevents grinding.

PETG: needs very little retraction (0.5–1mm direct) — over-retracting causes stringing due to oozy nature. Use more wipe distance instead.
5. Max Volumetric Speed Test

Determines the maximum rate your hotend can melt and push filament. This is the hard ceiling for all speed settings — go faster and you get under-extrusion. Especially critical for high-speed printers (150mm/s+).

How It Works

  • Prints a column increasing speed continuously
  • Find the height where under-extrusion first appears (gaps, rough surface, clicking extruder)
  • Set that volumetric value (mm³/s) in your filament profile
  • OrcaSlicer uses this to cap all speed settings automatically

Typical Values (mm³/s)

HotendPLAPETG
Stock E3D V610–148–11
Bambu stock18–2414–18
Dragon HF / Rapido25–3518–25
Volcano20–3015–22
6. First Layer Calibration

The foundation of every print. Correct first layer = parts that stick and release cleanly with good dimensional accuracy.

Live Z Offset

  • Use OrcaSlicer's built-in squish pattern (Calibration > First Layer)
  • Adjust Z offset in real-time while printing: turn negative = closer to bed (more squish), positive = farther
  • Correct: lines slightly squished together, no gaps, no ridges between lines
  • Too close: nozzle drags, plastic squirts sideways, nozzle may clog
  • Too far: lines round on top, gaps between lines, poor adhesion

First Layer Visual Reference

Lines should look like they have been gently pressed — slightly wider than the nozzle diameter, with adjacent lines just barely touching. A slight sheen from compression is correct.
Re-run first layer calibration after: nozzle change, hotend rebuild, bed surface replacement, significant temperature change in the room.
7. VFA — Vertical Fine Artifacts Test

VFAs (Vertical Fine Artifacts) are periodic ripples or bands on the vertical walls of prints caused by resonance at specific speeds — typically motor step frequencies or motion system resonance frequencies.

How to Run

  • Calibration menu > VFA
  • Prints a smooth-walled cylinder at incrementally increasing speeds per band
  • Identify bands with rippling — note the speed range
  • Set outer wall speed to avoid those ranges in your profiles

Common Causes & Fixes

  • Belt resonance — retension belts to the same pitch (~110 Hz on Bambu, ~100 Hz on CoreXY)
  • Motor frequency — avoid printing outer walls at the resonant speed (typically 50–80 mm/s on many printers)
  • Input shaper (Klipper) — run resonance compensation to eliminate VFAs altogether
VFAs are purely cosmetic but very visible on smooth-walled models. If you see regular banding at consistent spacing, VFA speed is the culprit.

Filament Profiles & Multi-Material

PLA (Polylactic Acid)

Print Temp190–220°C
Bed Temp55–65°C
Fan Speed100%
EnclosureNot required
Easy to print, low warp, brittle, not heat resistant — softens at ~60°C. Best for prototypes, display models, and non-functional parts.

Bed surfaces: PEI textured (excellent adhesion), glass, BuildTak.

Drying: Not critical but dry 4 hr at 45°C if you notice clicking sounds or bubbles (moisture symptoms).

PLA+ / PLA Pro

Print Temp205–230°C
Bed Temp60–70°C
Fan Speed100%
Tougher than standard PLA, slightly more flexible, better layer adhesion. Otherwise uses the same easy workflow as PLA.

PETG (Polyethylene Terephthalate Glycol)

Print Temp230–250°C
Bed Temp70–85°C
Fan Speed30–50%
EnclosureNot required
PETG sticks extremely well to PEI — use a glue stick or let the bed cool fully before attempting removal to avoid damaging the surface.

Strengths: Strong, flexible, chemical resistant, food-safe (with appropriate settings), heat resistant to ~80°C.

Stringing: Prone to stringing — lower temp, enable good retraction, and tune Pressure Advance.

Drying: Moisture sensitive — dry 6 hr at 65°C before printing.

Best for: Functional parts, containers, outdoor use.

ABS (Acrylonitrile Butadiene Styrene)

Print Temp230–250°C
Bed Temp100–115°C
Fan Speed0% (or minimal)
EnclosureREQUIRED
ABS emits fumes — always print in a well-ventilated area or use a HEPA/activated carbon filter. An enclosure is mandatory to prevent warping and layer cracking.

Strengths: Strong, machinable (drill, tap, sand, acetone-smooth), heat resistant to ~100°C.

Bed: PEI smooth at 80°C, or ABS slurry (dissolve ABS scraps in acetone and brush onto bed).

Drying: Dry if you notice moisture symptoms.

ASA (Acrylonitrile Styrene Acrylate)

Print Temp240–260°C
Bed Temp90–110°C
Fan SpeedMinimal
EnclosureREQUIRED
UV-resistant and weather-resistant — the best choice for outdoor prints. Behavior is similar to ABS but with superior UV stability.

TPU — Flexible (Thermoplastic Polyurethane)

Print Temp220–240°C
Bed Temp40–60°C
Fan Speed50%
Max Print Speed20–35 mm/s
Direct drive extruder strongly recommended. Bowden setups cause TPU to buckle inside the tube. Use no retraction or minimal retraction (0.5mm direct drive).

Properties: Flexible, rubber-like, chemical resistant. Shore hardness ranges from 85A (very soft) to 98A (semi-rigid).

Best for: Phone cases, gaskets, handles, flexible hinges.

PA / Nylon

Print Temp240–280°C
Bed Temp70–90°C
Fan Speed20–40%
Nylon is extremely hygroscopic. It MUST be dried 12 hr at 70°C and printed directly from a sealed dry box. Wet nylon prints very poorly.

Strengths: Strong, abrasion resistant, slightly flexible. Excellent for gears and mechanical parts.

Warping: Warps on large prints — use an enclosure, high bed temp, and a brim.

Nozzle: Abrasive — use a hardened steel nozzle.

PC (Polycarbonate)

Print Temp260–310°C
Bed Temp90–120°C
Fan Speed0–20%
EnclosureREQUIRED
PTFE-lined hotends are not rated above 260°C — an all-metal hotend is required for PC. Warping is severe; use glue stick with a 120°C bed.

Strengths: Strongest common filament. Impact resistant. Heat resistant to ~130°C.

Carbon Fiber Filled (CF-PLA, CF-PETG, CF-Nylon)

CF filaments are highly abrasive and will destroy a brass nozzle within a few hours. A hardened steel nozzle is mandatory.

Benefits: Higher strength, lower weight, very stiff, matte surface finish.

Settings: Follow the base material settings (PLA, PETG, or Nylon) — only the nozzle requirement differs.

Multi-Material Printing (AMS / MMU)

Flush volumes: The amount of filament purged when switching colors. Higher = cleaner transition, more waste. Tune per color combination.

Wipe tower: Purges old color between tool changes. Size should match your flush volume setting.

Avoid flush (purge into infill): Reduces waste and time but slows the print slightly.

Color order tip: Print light colors before dark within the same object to minimize contamination.

PLA + TPUVery difficult
Support interface trick: use PETG as the interface material when printing PLA parts. PETG won't bond to PLA, so supports release cleanly with no surface damage.

Speed & Performance Tuning

Speed Settings Hierarchy

Outer settings cap inner ones. Set from slowest (quality-critical) to fastest (hidden):

Outer wall30–60 mm/s — quality critical, keep slow
Inner wall100–200 mm/s — hidden, can be fast
Infill150–300 mm/s — hidden, max speed
Top surface80–120 mm/s — visible, moderate
First layer20–30 mm/s — critical for adhesion
Small perimeters50% of outer wall speed

Acceleration Settings

Normal printing3,000–5,000 mm/s²
First layer500–1,000 mm/s²
Outer wall3,000–5,000 mm/s²
Infill8,000–15,000 mm/s²
Travel10,000–20,000 mm/s²
Jerk5–10 mm/s (quality) / 20+ mm/s (speed)

Lower jerk = smoother direction changes but slower overall movement. Tune based on your quality vs. speed priority.

Input Shaping / Resonance Compensation

Reduces ringing and ghosting artifacts that appear at high print speeds. Requires an accelerometer or manual frequency testing to find your printer's resonant frequencies.

MZVLow vibration — good all-rounder
EIHigh speed tolerance
2HUMP_EIVery aggressive — maximum speed
In OrcaSlicer, enter the X and Y resonance frequencies found from your calibration run. Each algorithm handles the frequency differently — start with MZV.

Pressure Advance & Speed Interaction

Higher print speeds require higher PA values. Always calibrate Pressure Advance at the actual target print speed — a PA calibrated at 50 mm/s will be wrong at 200 mm/s.

High-Speed Printing Checklist

  1. Run the Max Volumetric Speed test — never exceed your hotend's capability
  2. Calibrate Pressure Advance at your target print speed
  3. Enable Input Shaping if your printer/firmware supports it
  4. Use Arachne wall generator (handles high speeds better than Classic)
  5. Increase fan speed — better cooling is needed at higher speeds
  6. Verify first layer adhesion — high acceleration can dislodge prints mid-run

Advanced Features

Variable Layer Height

Right-click an object → Variable Layer Height. A height map appears along the model.

  • Drag down on curved or detailed areas → thinner layers, more detail
  • Drag up on flat areas → thicker layers, faster printing
  • Adaptive mode: OrcaSlicer auto-suggests layer heights based on surface angle
Variable layer height can significantly cut print time on models with both flat and complex surfaces while maintaining detail only where needed.

Modifier Meshes

Add → Modifier Mesh → choose Box, Cylinder, Sphere, or load a custom STL. The modifier region can have completely different settings from the rest of the model.

Use cases:

  • Reinforce a screw boss area with 80% infill while the rest of the model uses 15%
  • Print a logo area at 0.1mm layers for detail while the rest prints at 0.2mm
  • Raise temperature in a region with poor layer adhesion

Troubleshooting Complete Guide

Stringing

Causes: Temperature too high, insufficient retraction, low travel speed, moisture in filament.

Fixes:

  • Lower nozzle temp 5°C at a time until stringing stops
  • Increase retraction distance (run the Retraction calibration tool)
  • Enable "Avoid Crossing Perimeters"
  • Increase travel speed
  • Dry your filament if the problem persists

Layer Separation / Delamination

Causes: Temperature too low, layer height too large, printing too fast, excessive cooling on ABS/ASA/Nylon.

Fixes:

  • Raise nozzle temperature 5–10°C
  • Reduce layer height (stay within 75% of nozzle diameter max)
  • Slow down print speed
  • Reduce fan speed for engineering materials (ABS/ASA/Nylon need minimal cooling)

Warping / Lifting Corners

Causes: Thermal contraction, poor bed adhesion, insufficient brim.

Fixes:

  • Increase bed temperature
  • Add a brim (5–10mm works well for most warping issues)
  • Use an enclosure for ABS / ASA / Nylon
  • Apply a glue stick to the bed surface
  • Clean the bed with IPA before every print
  • Verify Z offset — first layer must be properly squished

Under Extrusion (Gaps, Weak Layers, Thin Walls)

Causes: Clogged nozzle, speed too high, flow rate too low, temperature too low, worn PTFE, moisture in filament.

Fixes:

  • Run Flow Rate calibration
  • Perform a cold pull / atomic pull to clean the nozzle
  • Reduce print speed
  • Increase nozzle temperature
  • Inspect Bowden tube for gaps at the hotend connection
  • Dry filament if moisture is suspected

Over Extrusion (Blobby Surfaces, Elephant Foot, Thick Lines)

Causes: Flow rate too high, Pressure Advance too low.

Fixes: Reduce the flow ratio, calibrate Pressure Advance, increase retraction slightly.

Z Banding (Horizontal Lines Repeating at Regular Intervals)

Causes: Bent or misaligned Z lead screw, Z rod binding, Z backlash, loose Z coupling.

Fixes:

  • Check the Z coupling — a loose coupling causes vibration at each revolution
  • Check lead screw alignment (should be perfectly vertical)
  • Lubricate the lead screw with PTFE/super-lube grease
  • Check for any binding along the Z travel

Ghosting / Ringing (Echo of Features on Surface)

Causes: Printer resonating at the current print speed — vibrations echo through the frame.

Fixes:

  • Reduce outer wall print speed
  • Tighten all frame bolts to increase rigidity
  • Enable Input Shaping / Resonance Compensation
  • Change speeds to avoid the printer's resonant frequency

Poor Overhang Quality

Causes: Insufficient cooling, speed too high, overhang angle too steep (>45–50°).

Fixes: Run fans at maximum on overhang regions, slow outer wall speed, add supports, or redesign the model to keep overhangs under 45°.

First Layer Issues

Not sticking (too high)Lower Z offset (more negative) · slow first layer
Nozzle scraping (too low)Raise Z offset
Inconsistent / wavyRun mesh bed leveling · check for bed debris

Clogged Nozzle

Cold pull (atomic pull): Heat to print temp, push filament through, cool to 90°C (PLA) or 120°C (PETG), then pull firmly. The plug brings debris out with it. Repeat until the plug comes out clean.

Needle cleaning: Use a 0.3mm acupuncture needle to clear the nozzle tip while hot.

Flush method: Heat to 260°C and push fresh filament through to flush out degraded material.

Blob on Print Start

Causes: Filament oozes during heatup and drops onto the first layer when the Z hop releases.

Fixes: Add a prime / purge line in your start G-code, use a wipe-on-purge-block move, or enable wipe-before-print in OrcaSlicer.

Practice Projects — 6 Calibration-to-Print Challenges

Project 1 — Your First Print: Calibration Cube

Goal: Learn the full workflow and verify printer calibration.

  1. Download the XYZ Calibration Cube (free on Printables / Thingiverse — 20×20×20mm)
  2. Import into OrcaSlicer and center on the plate
  3. Settings: 0.2mm layers · 3 walls · 15% grid infill · 60°C bed · 210°C PLA
  4. Slice and step through all layers in the preview — check for issues before printing
  5. Export to printer and print
  6. Measure result with calipers: X, Y, and Z should each be within ±0.3mm of 20mm
  7. If X/Y is off: check steps/mm calibration in firmware
  8. If Z is off: check Z steps or layer height setting
  9. If walls have gaps: flow rate too low — run flow calibration
  10. If walls overlap: flow rate too high — reduce and re-calibrate
Skills learned: Import, basic settings, slice, preview, export, print, measure with calipers.

Project 2 — Retraction & Stringing Test

Goal: Dial in perfect retraction for your filament and setup.

  1. Download a retraction test tower (multi-spire model, different temperatures per section)
  2. Import · set 0.2mm layers · no supports · 2 walls
  3. Set temperature range in filament settings or use a temperature tower script
  4. Print and examine: find the section with least stringing and good layer quality
  5. Set that temperature as your filament's default
  6. Run OrcaSlicer's built-in Retraction calibration (Calibration menu → Retraction)
  7. Set retraction distance to the best-performing value
  8. Print again to verify — spires should have minimal stringing
Skills learned: Retraction tuning, temperature effect on stringing, calibration tools.

Project 3 — Functional Phone Stand

Goal: Print a useful object with proper orientation and structural strength.

  1. Download a phone stand STL from Printables (search "adjustable phone stand")
  2. Import and check orientation — base must be flat on bed with no unnecessary overhangs
  3. Settings: 0.2mm layers · 4 walls · 25% gyroid infill · 3 top/bottom layers
  4. Check for supports: if the device slot angle exceeds 45°, use Support Painting there only
  5. Add a 5mm brim for better adhesion on tall, thin parts
  6. Slice and check the preview layer by layer for any issues
  7. Note the estimated print time and material weight before printing
  8. Print, remove supports, and test with your phone
  9. If layers split: temperature too low — increase 5°C and reprint
Skills learned: Support painting, brim, strength-focused settings, functional part workflow.

Project 4 — Multicolor Print (Single Extruder Color Change)

Goal: Print a 2-color logo or text sign without AMS/MMU.

  1. Design or download a 2-layer text/logo model (base layer + raised letters = 2 separate heights)
  2. Import into OrcaSlicer
  3. Right-click the layer slider in preview → Add Color Change at the layer where the letters begin
  4. Slice and preview — the color change marker should be visible in the layer slider
  5. Export and print
  6. When the printer pauses: retract old filament, insert new color, purge until new color appears solid
  7. Resume the print
Skills learned: Filament change at layer, color planning, proper purging technique.

Project 5 — Functional Bracket (Strength-Focused)

Goal: Print a wall-mount bracket that can hold 2 kg.

  1. Design or download an L-bracket or shelf bracket (100×100×30mm, ~5mm thick)
  2. Orient correctly: vertical wall of the bracket flat on the bed — this maximizes layer-to-layer bond strength in the load direction
  3. Settings: 0.2mm layers · 5 walls (critical for strength) · 40% gyroid infill · 4 top/bottom layers
  4. No supports needed with correct orientation
  5. Slice and verify: switch to the wall preview mode and confirm 5 perimeter lines are visible
  6. Run Flow calibration before printing — accurate flow is critical for structural parts
  7. Print in PETG (stronger and more flexible than PLA under mechanical load)
  8. After printing: perform a load test — hang progressively heavier items to verify
Skills learned: Orientation for strength, PETG settings, wall count importance, flow calibration for structural parts.

Project 6 — Miniature / Detail Print

Goal: Maximum quality print for fine detail — miniature figure or decorative model.

  1. Download a detailed miniature STL (gaming miniature or decorative figure — standard size: 28–32mm tall)
  2. Import and scale if needed
  3. Settings: 0.1mm layer height · 3 walls (Arachne) · 20% infill · Variable Layer Height ON
  4. Supports: Tree supports · 0.2mm Z distance · 2 interface layers · 30° angle threshold
  5. Speeds: outer wall 25 mm/s · inner wall 60 mm/s · infill 100 mm/s
  6. Temperature: 1–2°C lower than your normal setting (reduces stringing on tiny features)
  7. Fan: 100% (best detail quality and bridging)
  8. Print and examine under magnification — assess detail capture quality
  9. If stringing on fine features: lower temp 2°C more and reduce support Z gap slightly
Skills learned: Variable layer height, tree supports, quality-focused settings, fine detail printing workflow.
