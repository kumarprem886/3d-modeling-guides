HueForge — Complete Guide
Photo-to-3D Multi-Color Filament Painting

What is HueForge

Info
HueForge converts images and photos into 3D-printable multi-color stacked filament prints.
Info
Core concept: each filament has a Transmission Distance (TD) value — darker/lower TD = blocks more light.
Info
HueForge stacks layers of different colored filaments to mix colors optically — like pointillism with plastic.
Info
Output: STL file + layer-change G-code instructions for your slicer.
  • Best printer typesBambu Lab (AMS), Prusa MMU, Palette, single-color with manual swaps
  • Best subject matterImage plaques, portraits, logos, art prints
  • Result typeWall art — all from standard FDM filament, no painting needed

Interface & Image Import

  • Import imageFile → Import Image (JPEG, PNG, SVG)
  • Image adjustmentsCrop, brightness, contrast, saturation sliders
  • Print widthSet mm width of final print (150–200mm for wall art)
  • Print depthTotal height in mm (typically 2–4mm for plaques)
  • Print resolution0.4mm minimum detail with 0.4mm nozzle
  • PreviewReal-time colored preview updates as you adjust
Tip
High-contrast images work best — portraits, logos, and nature scenes with clear light/dark separation.
Warning
Check your nozzle size matches the detail resolution — a 0.6mm nozzle cannot reproduce 0.4mm details.

Filament Library & TD Values

  • Low TD (0–10)Very opaque (white, grey) — blocks light completely
  • Mid TD (10–40)Semi-transparent — layers blend colors
  • High TD (40+)Transparent/translucent — light passes through
  • Add filamentsFilament Manager → Add Filament → set brand, color, TD value
  • Filament packsDownload from HueForge Discord or website
  • Built-in brandsBambu, Prusament, Polymaker, and others pre-loaded
  • TD calibrationPrint TD calibration card → measure → enter value
Tip
Start with 4–6 filaments maximum — fewer colors = simpler print with less risk of color change issues.
Warning
Always calibrate TD for your exact filament batch — TD can vary between colors and even batches of the same brand.
Info
Community filament packs on HueForge Discord have pre-measured TD values for hundreds of filaments.

Optimization & Export

  • Filament stack orderHueForge auto-calculates optimal stacking
  • Dithering typeFloyd-Steinberg (detailed) or Ordered (cleaner)
  • Problem areasRed highlights = poor color match zones
  • Slicer layer height match0.1 / 0.12 / 0.15 / 0.2mm — must match slicer exactly
  • Export formatSTL + color change G-code list, or separate STL per filament
Tip
Toggle each filament layer on/off in the preview to inspect its coverage before exporting.
Warning
The layer height in HueForge and your slicer MUST match exactly — even 0.01mm difference causes color misalignment.
Info
Reduce number of active filaments and preview result — sometimes 4 filaments looks nearly as good as 8 with less complexity.

Slicer Integration

  • PrusaSlicerImport STL, add filament color changes at layer heights (right-click layer timeline)
  • Bambu StudioImport STL with AMS filament assignments or color painting mode
  • CuraInstall Filament Change plugin → add pauses at each layer from HueForge list
  • OrcaSlicerBest Bambu integration, color changes via pause G-code
  • Print speed50% normal speed for quality on detailed images
  • First layer filamentUse same as bottom of stack (usually darkest)
Tip
OrcaSlicer or Bambu Studio give the cleanest integration with AMS — use them if you have a Bambu printer.
Warning
Double-check every layer change entry before printing — a single wrong layer number shifts all colors above it.

Workflow — Photo to Print

  1. Choose high-contrast image — portraits, logos, and nature scenes work best.
  2. Import to HueForge. Adjust brightness and contrast for best tonal range.
  3. Set dimensions: width (150–200mm for wall art), depth (2–3mm standard).
  4. Add your filament library. Match TD values by calibrating or using community values.
  5. HueForge suggests a filament stack. Enable/disable filaments, preview result live.
  6. Export STL + layer change list. Note the exact layer numbers for each color change.
  7. Import to slicer. Set exact matching layer height. Add filament pauses at listed layers.
  8. Print. Swap filament at each pause and resume. No post-processing needed.
Tip
Print a small test version (50mm wide) before committing to a full-size print — verify color accuracy and layer alignment.
Info
The wipe tower or purge block in your slicer is essential for clean color transitions — always enable it for HueForge prints.
