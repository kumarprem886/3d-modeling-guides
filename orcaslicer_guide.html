Workflow — Calibrate New Filament
  1. Install OrcaSlicer. Add your printer (select from list or manual setup).
  2. Create filament profile: set name, manufacturer, temperature range.
  3. Run Temperature Tower from Calibration tab → find optimal print temperature.
  4. Run Flow Rate calibration → adjust extrusion multiplier in filament profile.
  5. Run Pressure Advance calibration → set PA value in printer profile.
  6. Run Retraction calibration → set retraction distance in filament profile.
  7. Now slice actual models — fully calibrated settings give consistent, high-quality prints.
Tip
Save your calibrated filament profile with the brand and color in the name — build a library you can reuse for each spool.
Info
Re-run calibrations when switching to a new batch or color of the same filament — results can vary between batches.
