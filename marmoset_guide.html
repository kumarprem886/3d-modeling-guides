Portfolio Render Workflow

1
Import your final low-poly mesh with UVs unwrapped. Then import all PBR texture maps (Albedo, Normal, Roughness, Metalness, AO).
2
Create a new material and assign each texture to its correct slot — Albedo, Normal, Roughness, Metalness. Double-check color spaces (Albedo = sRGB, all others = Linear).
3
Place your HDRI sky light: pick an environment that matches the mood you want (studio grey for clean product shots, outdoor HDRI for naturalistic lighting).
4
Add 2-3 additional fill lights to control shadow darkness and add rim lighting. Use low-intensity directional lights or area lights for subtle bounce fill.
5
Set up the camera: adjust FOV (35mm is default, go lower for telephoto compression), and enable DOF if you are doing a close-up detail shot.
6
Add post effects: enable SSAO for contact shadows, add subtle Bloom if emissive elements are present, and add a gentle Vignette to draw focus inward.
7
Render high resolution: go to Scene → Render, set your output resolution, then press Ctrl + Shift + R to capture the final still.
8
Export a turntable video: go to Camera → Turntable, set rotation speed and frame count, then Capture to render the animated portfolio video.
Pro tip: Save your scene before rendering so you can tweak and re-render without losing your lighting setup. Name scenes with descriptive tags (e.g. "sword_studio_v02").
