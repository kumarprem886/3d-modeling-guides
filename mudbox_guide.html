Sculpt Layers & Subdivision Levels

Go to higher subdivision levelShift+D
Go to lower subdivision levelD
Add subdivision levelMesh → Subdivide
Add sculpt layer (non-destructive)Sculpt Layers panel
Adjust layer blend amountLayer opacity slider
Merge layer downRight-click → Merge Down
Protect areas from brush (freeze / mask)Freeze paint
Sculpt Layers work like Photoshop layers but for 3D geometry. Layer opacity blends sculpt intensity — great for subtle micro-detail control.
Always sculpt on a dedicated layer per detail type (pores, wrinkles, cloth). This keeps edits non-destructive and blendable.

PTEX format is natively supported in Maya and Arnold, removing the need to manage UV seams altogether.

Maya → Mudbox → Maya Pipeline

1
Model base mesh in Maya with correct topology and a clean UV layout
2
Send to Mudbox via File → Send to Mudbox from Maya (or import as OBJ)
3
Subdivide to 4–7 million polygons for fine surface detail
4
Sculpt detail passes: pores, wrinkles, scales, fabric weave — use separate layers per detail type
5
Paint diffuse, specular, and gloss maps in the Paint Room
6
Extract maps: Mesh → Extract Texture Maps → generate Displacement and Normal maps
7
Send back to Maya or export FBX with maps for rendering in Arnold
Keep your base mesh clean and quad-based. Mudbox's subdivision relies on clean topology — triangles and n-gons will cause pinching artifacts at higher subdivision levels.
