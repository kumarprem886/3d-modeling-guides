Hard Surface for Artists — Step by Step

InfoPlasticity is best for hard-surface mechanical shapes: vehicles, weapons, sci-fi props, product design. Its NURBS foundation produces mathematically perfect curves and fillets that polygon modeling can never match.
  • 1
    Concept Silhouette: Sketch the concept silhouette or gather reference images. Plasticity is best for hard-surface shapes with clear mechanical forms. Import a reference image as a backdrop if needed to trace proportions.
  • 2
    Block Primary Form: Use Shift+A to add primitive solids (box, cylinder, sphere). Combine them with boolean Union (J) and Subtract (K) to rough out the main shape. Work fast here — booleans are instant and reversible.
  • 3
    Add Secondary Forms: Draw NURBS curves (C) to define complex profiles and sweeps. Use Loft (select curves → L) to create surfaces between curves. Use Pipe (Shift+P) to run tubes/cables along drawn paths.
  • 4
    Fillet All Sharp Edges: Select edges and press F to fillet them. Drag to set the radius — Plasticity shows a live preview. Plasticity fillets are mathematically perfect NURBS curves, unlike polygon bevels that distort at corners.
  • 5
    Export to Target App: File → Export → OBJ for Blender/ZBrush, or STEP for Fusion 360/other CAD. For Blender, use the Plasticity Live Link plugin for real-time sync — update geometry in Plasticity and it appears instantly in Blender.
  • 6
    Texture & Detail: In Blender, UV unwrap the OBJ and add modifiers (bevel, subdivision). Import into Substance Painter for PBR texturing. For organic surface detail (scratches, wear), take to ZBrush and use ZBrush's surface noise and polish tools.
