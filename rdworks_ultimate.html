Understanding RDWorks & CO2 Lasers

RDWorks (also marketed as LaserWorks) is the official free design-and-control software made by Ruida Technology for their RDC series laser controllers. If you own a Chinese CO2 laser cutter — the kind that dominates the Indian market from ₹50,000 to ₹1,50,000 — there is a very strong chance your machine already runs a Ruida controller, and RDWorks is the software that talks to it. No licence fee, no subscription, no dongle. Free forever.

Why RDWorks dominates India: The entire affordable CO2 laser segment in India — sold by Cron Laser, Endurance, local Delhi/Surat/Mumbai resellers — ships Ruida-controller machines. RDWorks comes pre-installed or on a bundled USB stick. LightBurn costs USD 60–80 which equals ₹5,000–6,700, a significant added expense for a machine that cost ₹60,000. For many Indian users, RDWorks is "the" laser software.

The Ruida Controller Explained

The Ruida controller is a standalone embedded computer bolted inside your laser machine. It runs the motion system, fires the laser tube, reads the limit switches and manages the operator panel — all independently of your PC. Your computer only needs to be connected when you send a new job; once the file is transferred, you can disconnect and the controller runs the job entirely on its own from its internal memory.

Installation & First-Time Setup

Getting RDWorks talking to your machine is usually straightforward, but there are a handful of India-specific issues — power quality, summer heat stress on chillers, USB driver quirks on Windows 10/11 — that trip up new users. This section covers everything from install to first successful job.

Minimum (Works Fine)

  • 2GB RAM, 500MB disk space
  • USB 2.0 port (for controller connection)
  • 1024×768 display minimum
  • No graphics card requirements

  • Windows 10 64-bit (most stable)
  • 4GB+ RAM

  • SSD for faster file operations
  • Ethernet port for network connection

  • Linux (use LightBurn instead)

  • Windows RT or S-mode

Installation Steps

  1. Run as Administrator: Right-click the installer .exe → "Run as administrator". This is critical — RDWorks installs USB drivers that require admin rights. Regular user installation often fails silently.
  2. Accept the license and choose path: Default path C:\RDWorks\ is fine. Avoid paths with Chinese characters or very long paths (driver installer bug).
  3. Driver installation prompt: During setup, a separate "USB Driver" window will appear. Click Install. This installs the Ruida virtual COM port or HID driver. Do NOT cancel this step.
  4. Windows Defender / Antivirus warning: Some antivirus flags the driver installer as suspicious (false positive). Allow it. Add the RDWorks folder to your AV exclusions.
  5. Restart Windows: Required after driver installation for the USB device to be recognised properly.
  6. Launch RDWorks: First launch may be slow (30–60 seconds) as it initialises the work area. Subsequent launches are faster.

Connecting via USB

USB is the most common connection method for smaller machines. The Ruida controller appears as a USB device (not a COM port — it uses bulk transfer protocol).

  1. Power on the laser machine first, then connect the USB cable. Ruida controllers prefer this order — hot-plugging after PC boot can cause missed enumeration.
  2. Check Device Manager: Windows key → type "Device Manager" → look under "LibUSB-Win32 Devices" or "Universal Serial Bus devices" for "Ruida" or "RDC" device. If it shows with a yellow exclamation, driver didn't install — reinstall as Administrator.
  3. In RDWorks: Go to Config → Vendor Settings → USB. Connection type should auto-detect. Click "Read from machine" to test — if it returns machine parameters, you're connected.
  4. Test send: Draw a small shape (rectangle), click the Download button (↓), click "Send". The machine panel should beep and show the file name. If it does, USB is working perfectly.
USB cable quality matters in India: The stock USB cable that comes with most Indian-sold machines is often flimsy. If you get intermittent disconnects or "device not found" errors, replace with a quality shielded USB-A to USB-B cable (printer cable style), 1.5m maximum length. Long cables lose signal integrity.

Connecting via Ethernet (Recommended for Larger Machines)

Ethernet is more reliable than USB, especially for large files (detailed engravings). The Ruida controller has a built-in static IP address by default.

Ethernet Setup Procedure

  1. Default controller IP: 192.168.1.100 (check your controller panel: Menu → IP Config)
  2. Set your PC's network adapter to a static IP in the same subnet: e.g., 192.168.1.101, subnet mask 255.255.255.0
  3. Connect PC directly to controller with Ethernet cable (or via a router/switch — keep them on same subnet)
  4. In RDWorks: Config → Vendor Settings → Network → enter controller IP → test
  5. Ping test: open CMD, type ping 192.168.1.100 — if you get replies, network is good
Direct cable vs router: For a dedicated machine PC, connect directly with an Ethernet crossover cable (modern NICs auto-negotiate, so a straight-through cable works too). This avoids IP conflicts and gives maximum reliability. If you're on a shared office network, assign the controller a unique IP and ensure the gateway doesn't interfere.

India Power Setup — Critical Safety

CO2 lasers in India need stable power and proper grounding: These machines draw significant current (15–30A for 80W–130W machines), have high-voltage laser power supplies (15,000–25,000V), and water-cooled tubes. Poor power, bad grounding, or inadequate cooling causes tube failures costing ₹8,000–25,000 to replace.

⚡ Power Requirements

  • Voltage: 220–240V single phase (standard India supply)
  • Current: 40W machine ~8–10A; 80W ~15–18A; 130W ~25–30A
  • Use a dedicated circuit — don't share with heavy machinery
  • UPS/Stabiliser: A servo voltage stabiliser (₹3,000–8,000) is strongly recommended in areas with voltage fluctuation
  • Earthing: Proper 3-pin grounded outlet is mandatory. CO2 laser power supplies have capacitors that retain lethal charge.

🌡️ Summer Heat — Indian Climate Challenge

  • CO2 laser tubes need water cooling, typically 15–25°C
  • In Indian summer (40–45°C ambient), basic water pump+bucket setups fail fast
  • CW-3000 chiller (₹12,000–18,000): handles up to 80W in moderate climate
  • CW-5000 chiller (₹22,000–32,000): required for 100W+ or for Delhi/Rajasthan summers
  • Clean chiller water monthly — add a few drops of bleach or algae inhibitor to prevent biofouling in Indian humidity
  • Keep chiller in ventilated space — if chiller fan blows into a hot corner, its cooling capacity drops by 30–40%

Testing the Connection — First Test Job

  1. Draw a 20×20mm square in RDWorks canvas. Set layer to Cut, Speed 20mm/s, Power 15%.
  2. Click "Download" (the down-arrow button in the toolbar or in the right-side job panel).
  3. On machine panel: press Frame button — laser head should trace the bounding box without firing. Confirm it's positioned where you expect.
  4. Press Start on panel: Machine should move and cut. At 15% power on a 60W+ machine, this will barely mark acrylic — that's intentional for a connection test.
  5. If successful: Increase power for your actual material and run proper test grid (Tab 5 covers this).
Vendor Settings password: The default Ruida vendor settings password is RD8888. Some machines from Indian resellers are set to HF888, 888888, or 123456. If none work, contact your reseller — don't try to brute-force it as some firmware versions lock after failures.

Coordinate System & Origins

Understanding how RDWorks handles position is critical to avoid cutting in the wrong place.

Absolute Coordinates

Objects are positioned relative to the machine's home (origin) position. If your machine homes to top-right, the canvas (0,0) in RDWorks represents that corner. Use this when you need the job to cut in a specific fixed position on the bed every time — useful for jigs and fixtures.

Current Position

Job starts from wherever the laser head is when you press Start. Most commonly used mode. Position your laser head over the material corner, set that as origin on the controller panel, then send the job. Flexible but requires you to manually position each time.

Centre Mode

Job starts from the centre point of the work. Useful for engraving circular items (coasters, plates) on a jig — put the laser over the centre, set origin, and the job expands outward from there.

SVG Import Workflow (Most Common)

SVG from Inkscape is the most popular import path for Indian laser users. Inkscape is free, runs on Windows, and exports clean SVGs.

Inkscape → RDWorks: Step-by-Step

  1. Design in Inkscape with layers/colours matching your RDWorks layer convention (Red = cut, Black = engrave)
  2. Select all text → Path → Object to Path (converts fonts to curves — critical, RDWorks cannot use font files)
  3. Expand all strokes: If your cut lines have a stroke width, Path → Stroke to Path, then reduce outline stroke to 0.001mm (hairline). RDWorks cuts the centre of strokes.
  4. Remove fills from cut paths: Cut paths should be stroke-only, no fill. Fill on cut paths causes unexpected behaviour.
  5. Save as SVG (plain SVG, not Inkscape SVG — fewer compatibility issues): File → Save a Copy → Plain SVG
  6. In RDWorks: File → Import → select your SVG → click Open
  7. Verify scale: Check imported dimensions in the property bar against your design intent. SVG units can cause 3.78× scale error (96dpi vs 25.4mm px confusion) — if objects are wrong size, use Transform to scale by 25.4/96 = 0.2646
  8. Assign to layers: Objects should auto-assign by colour. If not, select objects and click the desired layer colour in the layer panel.
The 3.78× scale trap: Inkscape documents set to 96 DPI units export SVGs where 1 SVG user unit = 1 pixel = 0.2646mm. RDWorks sometimes interprets 1 SVG unit as 1mm, making your 100mm design import as 378mm. Always check after import. Fix: select all, use Transform → Scale → set width to correct dimension → apply.

DXF Import Workflow

DXF is the best format for precision work from CAD software. Use this when dimensions must be exact (mechanical parts, interlocking boxes, jigs).

  1. In your CAD software (AutoCAD, FreeCAD, Fusion 360, LibreCAD), ensure you're working in mm units.
  2. Export DXF: Use AutoCAD 2004/2007/2010 compatibility level — newer versions sometimes cause import issues.
  3. In RDWorks: File → Import → select DXF. A dialog asks for units — select mm.
  4. Check dimensions immediately: Select all, check property bar width/height against your design. DXF unit mismatch (inches vs mm) is common — if everything is 25.4× too large, you imported inches as mm. Delete, reimport, choose inch units, then use Transform to convert.
  5. Assign layers: DXF imports usually come in as one layer. Select all objects you want to cut → click red in layer panel. Select engrave objects → click black.

Raster Image Import (BMP/JPG/PNG)

For photo engraving on wood, bamboo, stone, glass — raster images are the input. The quality of your pre-processed image has enormous impact on engraving quality.

Image Preparation (Before Import)

  • Convert to greyscale: RDWorks works best with true greyscale (not RGB)
  • Increase contrast: Laser engravings have less tonal range than photos — boost contrast significantly
  • Sharpen: Unsharp mask helps recover detail that softens during engraving
  • Recommended DPI: 300–500 DPI at final size. 600+ DPI is wasted — laser spot size limits resolution
  • BMP format: Uncompressed BMP gives most predictable import. JPG is fine at high quality (90%+)
  • Tools: Photoshop, GIMP (free), Lightroom

Import & Layer Assignment

  • File → Import → select image
  • Image appears on canvas as a raster object
  • Assign to a layer — RDWorks will show "Scan" mode automatically for images
  • Resize using corner handles (hold Shift to constrain proportions)
  • In layer settings: choose dithering mode (Jarvis, Floyd-Steinberg, or Ordered)
  • Set scan interval (line spacing = effective DPI)
  • Lower scan interval = higher DPI = slower job

Setting Up Designs for Cut vs Engrave Using Layers

The fundamental rule: every object that will be cut must be a closed path with no fill, assigned to a layer set to "Cut" mode. Every area that will be engraved can be either a filled vector shape on a layer set to "Scan" mode, or a bitmap image.

Typical Multi-Operation Setup

Example: engraved wooden name plate with cut outline

  1. Black layer (Layer 1): The engraved text/graphic — set to Scan mode, 300mm/s, 40% power
  2. Red layer (Layer 3): The outer cut rectangle — set to Cut mode, 15mm/s, 75% power
  3. In layer panel, ensure Black layer order is above Red (engrave runs first, cut after)
  4. Enable both layers, disable any unused layers
  5. Send to machine — it will engrave the design, then cut the outline
India use case — Acrylic trophies: Design in CorelDRAW (still dominant in Indian trophy shops) → export DXF or SVG → import to RDWorks. Green layer = text engrave, Red layer = outer shape cut. Getting settings right is the difference between clean cuts and burnt disasters. Every combination of material type, material thickness, laser power, and focal length needs its own tested parameters. This tab gives you a solid starting framework, then explains every parameter in detail.

Scan+Cut Mode

Scan+Cut is a powerful single-layer operation where RDWorks first engraves the fill of a shape (scan mode), then cuts the outline of the same shape (cut mode), all in one layer pass. This is perfect for single-colour badge-style designs.

When to Use Scan+Cut

  • Small badges, labels, stickers where engrave+cut needed but design is simple
  • Rubber stamp making (engrave the design, cut the stamp outline)
  • Saves time vs managing two separate layers for simple designs

Limitation: Both engrave and cut use the same speed/power settings — you can't set cut power independently. For any complex job, separate layers are better.

Building a Power-Speed Test Grid

The test grid is your most important tool. Make one for every new material and every new tube.

How to Make a Test Grid in RDWorks

  1. Draw a grid of small rectangles (10×10mm each), 5 columns × 4 rows = 20 test squares
  2. Each column = different speed (e.g., 10, 15, 20, 25, 30 mm/s for cutting)
  3. Each row = different power (e.g., 50%, 60%, 70%, 80%)
  4. Assign all to one layer. Run the job.
  5. Examine results: find the fastest speed at which cuts fully penetrate → that's your optimal setting
  6. Save the test grid as a .rld template so you can rerun it quickly on new materials

Alternative: Use the RDWorks "Array" feature to duplicate shapes in a grid pattern with incremental layer property changes — faster than drawing manually.

Bi-directional Engraving Calibration

Bi-directional scan nearly doubles engraving speed but can cause "tearing" — offset between left-to-right and right-to-left passes. This shows as blurry or doubled text.

  1. Engrave a test pattern: a series of vertical lines or text at your working speed.
  2. Zoom in to check if lines from left passes align with lines from right passes.
  3. In RDWorks: Config → Machine Settings → Backlash (or Scan Offset) — adjust X-axis backlash in 0.1mm increments.
  4. Re-test. The correct value makes both scan directions land in the same position.
  5. This value changes with speed — calibrate at the specific speed you engrave at.

Working with Layers

Layers are the core of RDWorks' power. They let you run multiple operations — engrave photo, engrave text, cut score line, cut outline — all in a single job, in a controlled order, with independent settings for each. Mastering the layer system unlocks the full potential of your CO2 laser.

How the Layer System Works

In RDWorks, a layer is determined by object colour. Every object on the canvas has a stroke/outline colour. Whatever colour that is, the object belongs to the layer with that colour. There are 8 layers, each with a fixed colour (black, blue, red, green, orange, magenta, cyan, yellow). You cannot add more layers — 8 is the maximum.

Key concept: Layers in RDWorks are NOT like Photoshop layers (which control visibility and stacking of artwork). They are parameter groups. Two objects of the same colour both share the same speed/power/mode settings, even if they were drawn at different times. Change the layer settings → both objects change.

The 8 Layers — Full Details

Layer 2 Blue #0000FF Secondary engrave, lighter surface mark Scan
Layer 3 Red #FF0000 Main cut — outlines, through-cuts Cut
Layer 4 Green #00AA00 Score lines, half-depth cuts, fold marks Cut (lower power)
Layer 5 Orange #FFAA00 Perforation lines, dotted cuts Cut (with pauses / dash pattern)
Layer 6 Magenta #AA00AA Construction/reference lines (disabled or 0% power) Cut or disabled

Assigning Objects to Layers

Method 1: By Object Colour (Workflow Standard)

Design your artwork with cut lines in red, engrave areas in black (or whatever your colour convention is). When you import into RDWorks, objects automatically go to the matching layer based on their stroke colour. This is the cleanest workflow — design with intent baked in.

In Inkscape: Set stroke colour to pure #FF0000 for cuts, #000000 for engrave. No fill on cut paths.

Method 2: Select and Click Layer

Select any object(s) on the canvas, then click the desired layer colour swatch in the layer panel on the right. The object(s) immediately move to that layer and take its colour and settings.

Use this when you imported a file where everything ended up on one layer and need to reassign.

Layer Order — Execution Sequence

RDWorks executes layers from top to bottom in the layer panel list. You can drag layers to reorder them. The correct order for most jobs is:

Recommended Layer Execution Order

  1. First: Deep engrave layers — photo engravings, area fills. These should run before cutting so the material is still held in place.
  2. Second: Light engrave / surface mark layers — text, light graphics.
  3. Third: Score/crease lines — half-depth cuts for folding.
  4. Last: Cut layers — full through-cuts. Always cut last so parts stay in place during engraving.

Why this order matters: If you cut first, the cut piece shifts slightly (or falls through the honeycomb). Then the engrave pass runs over a moved piece — result is misaligned engraving. Engrave first, cut last is the golden rule.

Enabling and Disabling Layers

Each layer has a visibility/enable checkbox in the layer panel. Disable a layer to prevent it from running in the next job send — it stays in the file but won't execute.

Use Cases for Disabling Layers

  • Run only the engrave first on a test piece, check alignment, then enable and run the cut layer
  • Send a job to machine without construction/reference lines you use for layout only
  • Run partial production: disable layers that are already done if a job was interrupted
  • Test a multi-operation job in stages before committing full production run

Layer Output Toggle

In the layer panel, the checkbox column controls whether each layer sends to the machine. There's also a separate "Show" toggle (eye icon on some versions) for canvas visibility — this is different. You can have a layer visible on screen but disabled from output.

Tip: Use a reference/dimension layer in magenta with Output disabled — it helps with design layout but never accidentally fires the laser on your dimension lines.

Advanced Layer Techniques

Nested Cuts (Puzzle Pattern)

For interlocking parts, use one layer for inner cuts and another for outer cuts. Set inner cut layer first in execution order. This prevents inner pieces from shifting before outer cuts separate them from the sheet.

Two-Pass Deep Cutting

For thick acrylic (8–10mm on 80W machine): use two separate layers for the same cut path. First layer: 60% power, second layer: same path, 65% power. Between passes, the first kerf helps guide the second pass deeper without excessive heating.

Selective Output per Production Run

Have a base design with multiple text/name fields on different layers. For each customer order, enable only the name/text layer for that customer. This avoids redesigning for each piece and reduces risk of errors.

Layer as Jig Reference

Create a "jig" layer (disabled from output) showing the exact position of pre-cut material on the bed. Use this to place new cuts in correct registration — e.g., second-side engraving on a workpiece that was flipped.

What CO2 Laser CANNOT Do — Bare Metal

A CO2 laser (10,600nm wavelength) cannot engrave or cut bare steel, aluminum, copper, or brass. The metal surface reflects the CO2 beam instead of absorbing it — this wastes energy and can even damage your laser tube or optics.

Why this happens:

  • CO2 wavelength (10.6 micron) reflects off polished metal surfaces
  • Fiber laser (1064nm wavelength) is absorbed by metal — that's the correct tool
  • CO2 CAN mark metal using Cermark / LaserBond spray — apply spray, engrave, wash off excess. The bonded residue leaves a permanent black mark.
India Market: If customers ask for steel/aluminum trophy engraving → refer them to a fiber laser operator or quote with Cermark spray (+₹50–100/piece for spray cost). Fiber lasers start at ₹1.5 lakh in India — a natural upgrade when your business grows.

Jig Making for Repeatable Positioning

A jig is a template that holds your material in exactly the same position every time. Jigs are the #1 productivity tool for batch production.

How to Make an Acrylic Jig:

  1. Measure your product (e.g., coaster: 90×90mm, 10mm thick)
  2. In RDWorks, draw a rectangle 90×90mm and arrange your grid (e.g., 3×3 = 9 coasters)
  3. Add 1–2mm tolerance to each slot (so 92×92mm per slot)
  4. Cut the jig from 3mm acrylic — the cutouts become the holding slots
  5. Tape the jig to your bed at Machine Origin (Absolute mode)
  6. Drop in products, run batch job — repeat without re-aligning

For photo engraving, the laser can only be ON or OFF (no true grayscale). Dithering simulates grayscale using dot patterns. Choosing the right dither mode dramatically affects portrait quality.

  1. Open photo in Photoshop/GIMP → Convert to Grayscale

Mirror / Flip — Critical for Rubber Stamps

CRITICAL: Any design that will be stamped, transferred, or read through a material MUST be mirrored horizontally before engraving. Text engraved normally onto rubber will print backwards.

Pro Maintenance Tips

🔍
Clean mirrors weekly in India
India's dust, pollen, and particulate air quality means optics get dirty 3× faster than in temperate climates. A dirty mirror reduces power by 20–40% and causes uneven beams. Weekly clean = consistent output.
💧
Check water level daily
Chiller water evaporates, especially in dry seasons. Low water level means inadequate cooling → tube damage. Check every morning before starting. Top up with distilled water only.
💨
Never run without exhaust
Even for "just a quick test." Fumes from acrylic, rubber, and MDF are toxic. Exhaust must vent outside. Carbon filter alone is not sufficient for daily production — always use active exhaust to outside.
⚡
Use a servo stabilizer
India's power supply can fluctuate ±15%. A 2KVA servo voltage stabilizer (₹3,000–8,000) protects your machine electronics and ensures consistent laser output. Essential in Tier-2/3 cities with unreliable power.
🔧
Lens tissue only for optics
Never use paper towel, cloth, or kitchen tissue on mirrors and lenses. Only use proper lens tissue with 99% IPA. One scratch on a lens = ₹500–2,000 replacement. Wipe in one direction, not circular.
📁
Maintain a test log book
Keep a physical notebook with material tests: date, material, supplier, settings, result (photo). This becomes invaluable over months — you'll never need to re-test the same material twice.
🌡️
Warm-up the machine
Run a 5-minute warm-up at 10–15% power before starting production jobs. CO2 tubes perform more consistently after thermal stabilization. Skip warm-up = first few pieces may have slightly different output.
📐
Focus is everything
1mm out of focus = 20–40% power loss at the material surface. Re-focus every time you change material thickness. Make 5 acrylic focus gauges for your common thicknesses (3mm, 5mm, 6mm, 10mm, 15mm) and label them.
