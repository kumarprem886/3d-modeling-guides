Sculpting Tools

Core Brushes

  • Draw (default) — Push the mesh surface outward. The primary building brush for adding volume.
  • Smooth — Flatten and even out surface irregularities. Also activate by holding Shift while using any other brush.
  • Grab — Click and drag to pull large areas of mesh. Great for establishing major forms quickly.
  • Pinch — Pull surface edges tighter. Creates sharp ridges and fine definition.
  • Flatten — Press the surface down to align with the local average plane.
  • Inflate — Push mesh outward uniformly in all directions, like inflating a balloon. Good for puffing up lips, cheeks.
  • Crease — Create sharp indented lines. Perfect for nostrils, ear lines, eyelids, wrinkles.

Brush Controls

  • Brush size — Right slider in the toolbar, or scroll the mouse wheel (if not zooming)
  • Brush strength — Left slider in the toolbar
  • Invert brush — Hold Alt while painting to push inward instead of outward (works on all brushes)
  • Smooth while sculpting — Hold Shift to temporarily switch to Smooth brush on any tool
Dynamic Tessellation — The Key Feature
Dynamic Tessellation (DynaTess) is Sculptris's defining feature. Unlike traditional sculpting apps, you never manually subdivide — the mesh automatically gets more triangles exactly where you paint, and fewer where you don't.

How DynaTess Works

  • Sculptris adds new triangles automatically under your brush as you paint in fine detail
  • Areas you haven't touched stay low-poly — efficient memory usage
  • No need to manage subdivision levels, layers, or LODs
  • Enable/disable — checkbox labelled "DynaTess" in the brush settings panel (on by default)
  • Detail slider — Controls how fine the tessellation gets. Higher = more triangles = smoother curves = slower performance

Managing Mesh Resolution

  • Reduce mesh — Filter → Reduce Selected (decrease poly count when sculpting is done, for export)
  • Import mesh — File → Import Mesh (OBJ format only)
  • Export mesh — File → Export (OBJ format only)

ZBrush Pipeline

  • Block in major forms in Sculptris (fast and intuitive)
  • File → Export as OBJ when rough form is ready
  • Import OBJ into ZBrush → subdivide → add production-level microdetail (pores, wrinkles, fabric)
  • This is the classic beginner-to-pro pipeline used by many artists
Materials & Render

Matcap Materials

  • Material preview sphere — Right panel. Click it to cycle through built-in Matcap lighting presets.
  • Matcaps (Material Capture) simulate full studio lighting in a single texture — great for checking form.
  • Custom Matcap — Drag any image file onto the material sphere to use it as a custom matcap.
  • Download free Matcaps from ZBrush Central or Pixologic's resource library.

Scene & Rendering

  • Background color — Change in Scene settings panel
  • No dedicated render output — Sculptris has no renderer. Use OS screenshot (Win+Shift+S) to capture viewport.
  • For proper rendering — File → Export OBJ → import into Blender → apply materials and render with Cycles or EEVEE

Symmetry

  • Enable radial X symmetry via the Symmetry checkbox in the toolbar
  • Sculpt one side of a face or character, the other side mirrors automatically
  • Rotate symmetry plane with Ctrl+Shift+drag if your model is off-axis

File Operations

  • File → New — Start fresh with a new default sphere
  • File → Import Mesh — OBJ format only. Import any existing mesh to continue sculpting.
  • File → Export — OBJ format only. Always export as OBJ for use in other apps.
  • Important: Sculptris only supports OBJ. No FBX, no STL, no GLTF. Export OBJ and convert in Blender if needed.
Workflow — Beginner's First Sculpt
This workflow walks through sculpting a basic head/face from scratch in Sculptris, then exporting for use in other software.
1
Open Sculptris — A sphere appears automatically. This is your starting point. Every sculpt begins from a primitive shape.
2
Enable Symmetry — Check the X axis symmetry checkbox in the toolbar. This mirrors every brushstroke so both sides of the face are sculpted simultaneously.
3
Block major forms with Grab — Use Grab tool with a large brush to pull out: nose area, eye socket indentations, chin. Establish proportions before adding any detail.
4
Build volume with Draw — Switch to Draw tool to build up: cheeks, brow ridge, lips, forehead. Hold Alt to push inward for concave areas like eye sockets.
5
Define edges with Crease — Use Crease tool for sharp definition: nostril edges, ear canal, eyelid crease, lip line, philtrum groove. Reduce brush size significantly for this step.
6
Smooth and refine — Hold Shift to smooth out any lumpy areas. Then go back with a smaller Draw or Pinch brush for refined surface detail. Iterate smooth → refine → smooth.
7
Export and continue — File → Export as OBJ. Import into ZBrush for production-level microdetail and texturing, or import into Blender for rigging, rendering, or 3D printing prep.
