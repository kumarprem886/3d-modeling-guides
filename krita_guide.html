Workflow / Concept Art for 3D

1
Set canvas to 4K (4096×4096) for texture work, 1920×1080 for concept art
2
Create base composition with big shapes using large brushes — value (light/dark) first
3
Add layers: Multiply mode for shadows, Screen mode for highlights
4
Refine details with smaller brushes, use Smear brush to blend transitions
5
Use Reference Images panel to keep reference visible while painting
6
For textures: enable Wrap Around mode, paint and see tiles live
7
Export PNG for concept art handoff, or export PBR channel maps for 3D use
