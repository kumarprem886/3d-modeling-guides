🔍 What is EzCad?

EzCad is the laser control software developed by JCZ Technology (Beijing Jike Shengong Co.) and bundled free of charge with the vast majority of Chinese fiber laser marking machines. It acts as the complete brain between your PC and the galvo scanner — controlling every laser parameter from pulse width to scan speed.

India Context: Almost every fiber laser machine purchased from IndiaMART, Alibaba, or local machine dealers in India (₹40,000–₹1,50,000 range) ships with EzCad2 preloaded on a CD or USB, along with the USB dongle required to run it.

🎛️ What Does EzCad Control?

EzCad communicates with the galvo controller card (typically BJJCZ board) inside your machine and manages:

  • Laser Power % — fraction of max wattage delivered
  • Frequency (kHz) — pulses per second
  • Pulse Width (ns) — MOPA lasers only
  • Scan Speed (mm/s) — galvo mirror movement speed
  • X/Y Galvo Mirrors — position and path control
  • Red Dot Pointer — preview beam for positioning
  • Rotary Axis — if rotary attachment connected
  • I/O Signals — start trigger, footswitch, automation

NEVER lose or damage the USB dongle. Without it, EzCad will not run at all. It is hardware-locked. Replacement costs ₹3,000–₹8,000 and takes 1–3 weeks to source in India.

📦 Installation — Step by Step
1
Get the EzCad installer Use the CD or USB that came with your machine. If lost, download EzCad2 from jcz.com.cn (the official JCZ website). Make sure the version matches your controller card generation.
2
Run installer WITHOUT dongle plugged in This is critical. Plug in the dongle only after the software and drivers are fully installed. Plugging in too early causes driver conflicts in many Windows versions.
3

6
Connect the laser machine Via USB cable (most machines) or PCIe card (some industrial machines). The connection is to the BJJCZ controller board inside the machine, not the laser source directly.

⚠️ Dongle Troubleshooting
"Dongle Not Found" Error
  1. Unplug dongle, wait 10 sec, re-plug
  2. Try a different USB port (use USB 2.0, not USB 3.0 if possible)
  3. Try plugging through a USB 2.0 hub
  4. Uninstall dongle driver in Device Manager, reinstall
  5. Check if another program is using Sentinel drivers
  6. Run EzCad as Administrator
Blue Screen / Driver Conflict
  1. Boot Windows in Safe Mode
  2. Open Device Manager, find the dongle under USB or "Other Devices"
  3. Right-click → Uninstall (check "delete driver files")
  4. Reboot normally
  5. Re-install dongle drivers from CD
  6. If persists: try older driver version from JCZ website
EzCad Opens but No Laser Connection
  • Check USB cable from PC to machine controller board (not the laser source)
  • Try a different USB cable — the included cables are often low quality
  • Check Device Manager for the BJJCZ controller card — should appear without errors
  • Ensure machine is powered on before launching EzCad
  • Reinstall the controller card drivers (separate from dongle drivers)
💡 Windows Optimization for EzCad
  • Power Plan: Set to "High Performance" — Control Panel > Power Options
  • USB Sleep: Disable in power plan "USB selective suspend setting → Disabled"
  • Screen Saver: Disable completely — can interrupt long marking jobs
  • Windows Updates: Pause during production — forced restarts kill live jobs
  • Antivirus: Add EzCad folder to exclusions — some AV blocks dongle communication
  • Background Apps: Close Chrome, Teams etc. during marking — they cause micro-freezes
🇮🇳 India Power Supply Note
Use a voltage stabilizer (servo stabilizer) and UPS. Indian mains voltage fluctuates ±10–15% in many areas. The BJJCZ controller card is sensitive to power spikes. A ₹3,000–5,000 servo stabilizer protects your ₹60,000+ machine. The UPS gives you 5–10 minutes to finish a job during power cuts.

🔄 SVG → EzCad Workflow (Critical for Stamp Plates)

Since EzCad can't open SVG directly, here are three proven conversion methods:

Option A — Inkscape → DXF R14 (RECOMMENDED)
1
Open your SVG in Inkscape (free at inkscape.org)
2
Select all objects (Ctrl+A), then go to Path → Object to Path to convert any shapes/text to paths
3
Stroke to Path — if you have stroked outlines: Path → Stroke to Path. This makes outlines into filled regions
4
File → Save As → DXF R14 — in the DXF dialog, set units to mm, enable "Use ROBO-Master type of spline output"
5
In EzCad: File → Import → select your DXF → click Open → place on canvas
6
After import: go to Edit → Select All, then Modify → Explode if needed to separate grouped paths
Option B — Inkscape → PLT (HPGL)
  1. Open SVG in Inkscape
  2. Convert all to paths (Object to Path)
  3. Extensions → Export → Plot (or use the PLT export extension)
  4. Import PLT in EzCad — very reliable, curves stay smooth
Option C — Adobe Illustrator → AI Legacy Format
  1. Open or create in Illustrator
  2. File → Save As → Adobe Illustrator (.ai)
  3. In options: set compatibility to Illustrator 8 or earlier (legacy format)
  4. Import the .ai file in EzCad
Note: Modern AI format (CC) won't import. Must save as legacy AI 8 or older.

✅ Best Practices for Stamp Plate File Prep
Follow this checklist before importing any design for stamp plate work:
  1. Design at actual size in Inkscape (mm units) — what you design is what gets marked
  2. Convert all text to paths (Text → Object to Path) so fonts don't matter on the EzCad PC
  3. Expand all strokes to paths (Path → Stroke to Path) — EzCad doesn't handle stroke widths well
  4. Remove any fills — only outlines/paths should remain for vector marking
  5. Combine overlapping paths (Path → Union or Difference as needed)
  6. Keep node count reasonable — under 5,000 nodes for smooth operation
  7. Export as DXF R14, mm units
  8. Test import in EzCad before going to machine
⚙️ Accessing Mark Parameters

Mark parameters control exactly how the laser behaves when it marks. Access them by:

  • Method 1: Double-click any object on canvas → Mark Parameters dialog opens
  • Method 2: Double-click the color/pen in the Object List (right panel)
  • Method 3: F4 → Pen Settings tab
Parameters set on a pen/layer apply to all objects on that layer. Parameters set on an individual object override the layer. Best practice: set params per layer.
🎛️ All Key Parameters — Explained
SPEED mm/s — galvo mirror movement speed
Controls how fast the galvo mirrors sweep across the material. Higher speed = less time per mm = less energy delivered = lighter mark. Lower speed = more time per mm = more energy = deeper, darker mark.
SS304 Stamp Outline: 200–400 mm/s | Deep Engrave: 100–200 mm/s | Fast Surface Mark: 500–800 mm/s
POWER % — percentage of laser's maximum rated wattage
Sets what fraction of the laser's full power is delivered. A 20W laser at 60% delivers 12W. Higher power = more material removal per pass. Too high = burning edges, oxidation, rough surface. Too low = no mark or very faint.
SS304 Stamp: 50–70% | Annealing (color): 20–40% | Aluminum: 40–60% | Deep engrave: 70–90%
FREQUENCY kHz — pulses per second (FIBER LASER ONLY)
How many laser pulses fire per second. Lower frequency = fewer, more energetic pulses = deeper material removal per pulse. Higher frequency = more pulses, less energy each = smoother marks, better for surface-level work. This is one of the most important parameters to understand.
SS304 Stamp Plates: 20–50 kHz | Annealing/Color: 50–100 kHz | General marking: 30 kHz | Deep engrave: 20–30 kHz
Q-PULSE WIDTH ns — nanoseconds (MOPA FIBER LASER ONLY)
Only available on MOPA (Master Oscillator Power Amplifier) fiber lasers. Controls the duration of each laser pulse in nanoseconds. Narrow pulses (2–4ns) create very high peak power for fine detail and color marking. Wide pulses (100–200ns) have more total energy per pulse for material removal. Standard Q-switched fiber lasers have a fixed pulse width — only MOPA gives you this control.
Fine Detail / SS304 outline: 4–8 ns | Color marking on Ti: 2–4 ns | Deep engraving: 100–200 ns | General marking: 60–100 ns
Don't have this setting? Your machine is a standard Q-switched fiber laser, not MOPA. That's normal — most budget machines are Q-switched. MOPA machines cost more but give color marking capability.
LOOP COUNT passes — how many times the laser retraces the same path
Each pass adds depth. One pass may be enough for surface marking; 3–5 passes for deeper engraving. More passes also means more heat accumulation — add delays between passes for heat-sensitive materials.
Surface mark: 1 pass | Stamp plate outline: 1–3 passes | Deep engrave (hatch): 2–5 passes
JUMP SPEED mm/s — galvo travel speed when laser is OFF (moving between paths)
Speed at which galvo repositions between objects while the laser is off. Set this as high as possible (3000–8000 mm/s) for efficiency. If you get positioning errors or marks appearing where they shouldn't, reduce jump speed.
Default: 3000 mm/s | Maximum: 8000 mm/s | Troubleshoot positioning: reduce to 1000–2000 mm/s
START TC / END TC / POLY TC μs — timing correction delays
Time Correction values sync the laser firing with the galvo mirror position. START TC = delay before laser fires at start of a path. END TC = delay after laser stops at end of path. POLY TC = delay at corners/direction changes. Critical for corner quality. If corners are burned/rounded, adjust these values. Default factory values are usually fine.
Typical defaults: Start TC 150–300 μs | End TC 50–100 μs | Poly TC 50–100 μs
WOBBLE / WOBBLE AMPLITUDE mm — oscillation added to path
Adds a slight zigzag oscillation perpendicular to the scan direction. Creates a wider effective mark without changing speed or requiring multiple passes. Good for creating wider line marks in one pass.
Line widening: 0.05–0.2 mm amplitude | Texture effects: 0.1–0.5 mm

🔨 SS304 Stamp Plates — Overview

SS304 (304-grade stainless steel) is the most common material for rubber stamp metal plates in India. A laser-engraved SS304 stamp plate is used as the master die for making rubber/polymer stamps. The plate needs deep, clean, sharp edges — the quality of the engrave directly determines stamp quality.

Typical Stamp Plate Specs
  • Thickness: 0.3mm – 1.0mm SS304 sheet
  • Mark depth needed: 0.05mm – 0.15mm for rubber stamp use
  • Area: 30×30mm to 80×60mm typical
  • Feature size: Minimum 0.3mm line width recommended
  • Surface finish: 2B or BA (bright annealed) — smoother is better
Why Fiber Laser for SS304
  • 1064nm wavelength — absorbed excellently by steel
  • Galvo scan = very fast, precise
  • No tooling wear — same quality every plate
  • Can do 10mm×10mm plate in under 2 minutes
  • Extremely sharp corners — better than chemical etching
  • No chemicals, no mess, no darkroom
📐 Stamp Plate Design Workflow
1
Design in Inkscape (recommended) — Create your stamp design at actual size in mm. Use black for areas to be engraved (recessed). Text should be correct-reading (not mirrored) — EzCad will handle mirror if needed.
2
Prepare the file — Convert text to paths, expand strokes to paths, remove fills, keep only outlines. For stamp plates you engrave the OUTLINE of text/logo, then fill with hatch inside.
3

4
Set up layers in EzCad — Pen 1 (red): outline pass with outline parameters. Pen 2 (blue): hatch fill with hatch parameters. Assign objects to correct pens.
5

6

7

🔲 Hatch & Fill Modes

Hatch in EzCad is the equivalent of "fill" or "raster" in LightBurn — it fills a closed shape with parallel lines, turning an outline into a solid-looking engraved area. Understanding when to use hatch vs outline is one of the most important decisions in fiber laser work.

What Is Hatch?

Hatch = Parallel Lines Inside a Closed Shape

When you apply hatch to a closed vector shape (circle, rectangle, text outline, logo), EzCad fills the interior with a series of parallel laser passes spaced by the line spacing value. The galvo scans back and forth, removing or marking material across the entire filled area — like mowing a lawn in parallel strips.

  • Outline (Line mode): Laser traces only the edge path — fast, clean, minimal material removal
  • Hatch (Fill mode): Laser fills the interior — slower, removes more material, creates solid appearance

Hatch Parameters Explained

Line Spacing mm
Distance between parallel hatch lines. Smaller = denser fill, better coverage, longer time. Larger = visible lines, faster.
Stamp/jewelry: 0.02–0.04mm | General: 0.05–0.08mm | Fast/coarse: 0.1mm+

Hatch Types

Unidirectional

Laser fills in one direction only — travels back to start of each line before firing again. Slower but very consistent marking direction. Good for materials sensitive to directional marks.

Bidirectional (Zigzag)

Laser fills forward then back (zigzag). Fastest hatch mode — roughly 2x faster than unidirectional. Use for most SS304 and metal work. Minor difference in appearance.

Bow-Tie (X-Hatch Decorative)

Creates alternating diagonal fill — decorative pattern. Rarely used for functional engraving. Occasionally used for textured backgrounds on gifts.

Cross-Hatch (Two Pass)

Runs hatch at angle X, then runs again at X+90°. Creates a grid pattern. Removes significantly more material — good for deep engraving and background removal. Takes 2x as long as single hatch.

For Stamp Plates — Outline Only Is Best

Key insight: For rubber/polymer stamp production, you are just marking SS304 — the stamp plate itself uses the SS304 as a jig/registration surface and the laser depth just needs to be consistent. For Logo Engraving on Awards and Gifts

  • Thin decorative borders: Line mode

For Deep Engraving (Background Removal)

Heavy Hatch for Material Removal

This is slow but creates a professional deep-engraved effect. A 50×50mm background fill at 0.02mm spacing takes 10–20 minutes.

🔧 Troubleshooting — 20+ Problems & Fixes

The most common issues users face with EzCad and fiber laser machines, with specific attention to India-specific problems like power quality and climate.

Safety first: Never look into the laser aperture or at beam reflections. Always wear appropriate OD7+ fiber laser safety glasses (1064nm). The F2 key marks immediately — keep your workpiece in position before pressing.

Software / Connection Issues

1. Dongle Not Found / "USB Key Not Found" Error

Cause: EzCad2 requires its JCZ USB dongle to run. If dongle is not recognized, EzCad shows this error at startup and won't operate.

Fixes:

  • Try a different USB port (USB 2.0 preferred over USB 3.0 for dongles)
  • Check Device Manager for yellow exclamation on dongle — install Sentinel HASP driver manually
  • In monsoon season: humidity can cause poor USB contact — clean dongle pins with dry cloth, try compressed air in port
  • Dongle may be damaged — contact your India supplier for replacement. Keep the dongle serial number safe for JCZ registration
  • EzCad3 uses a different activation system — see Tab 12 for EzCad3 license issues
2. "Card Not Found" / "Controller Not Connected" Error

Cause: EzCad cannot communicate with the JCZ galvo controller card (internal PCIe card or external USB controller).

Fixes:

  • Check that the controller cable (DB25 / fiber cable from controller to laser) is firmly connected
  • For PCIe cards: ensure card is fully seated in slot — vibration in transit sometimes unseats it
  • Restart EzCad after reconnecting hardware
  • Windows update sometimes breaks the JCZ driver — reinstall driver from the CD that came with the machine
  • Try a different USB port for USB-based controllers (USB 2.0)
  • India-specific: power surge can damage controller — check if the card is getting 12V/5V power correctly

Marking Quality Issues

3. Marks Too Light / Not Visible

Possible causes:

  • Power too low — increase power % or slow down speed
  • Focus incorrect — workpiece not at focal length (typically 160mm or 254mm from F-theta lens). Test focus by marking at different Z heights
  • F-theta lens dirty — clean with lens tissue + IPA (isopropyl alcohol). Never touch lens surface with fingers
  • Fiber cable bent too tightly — check minimum bend radius (typically 100mm minimum) from laser to galvo head
  • Laser diode degrading — check output power with power meter if available
  • Material reflectivity — very polished surfaces mark lighter. Lightly sand or apply marking compound (Cermark / Brilliance)
4. Marks Too Deep / Burning / Material Warping

Possible causes and fixes:

  • Power too high — reduce power % or increase speed
  • Too many passes — reduce to 1 pass
  • Thin material — SS304 0.3mm will warp if power is too high; reduce by 20% and test
  • Speed too slow — increase speed first before reducing power
  • Frequency too low (MOPA) — low frequency = more energy per pulse = deeper marks. Increase frequency
5. Uneven Marking Across the Field (Edge Distortion)

Cause: Galvo field correction file (.cor file) is missing, corrupted, or wrong. The .cor file corrects for optical distortion at the edges of the marking field.

Fix:

  • Check that the correct .cor file for your lens (110mm, 160mm, 200mm field) is loaded in EzCad: File → Laser Settings → Correction File
  • If marks are fine in center but distorted at edges, the .cor file is wrong — request correct file from your machine supplier
  • Your machine supplier should have provided the .cor file on a USB stick or CD with the machine
  • JCZ field correction files are lens-specific and machine-specific — don't use another machine's file
6. Text / Design Appears Mirrored

Fix: In EzCad, File → Laser Settings → check "Mirror X" setting. Also check "Mirror Y". One combination gives the correct non-mirrored output for your specific galvo head orientation. Your machine supplier should tell you the correct mirror settings — it depends on how the galvo head is mounted (which direction X and Y scan).

Positioning and Accuracy Issues

7. Design Shifted from Expected Position

Causes and fixes:

  • EzCad origin not set correctly — use the red dot pointer to set the origin. Press F1 to move red dot to center of design, then use that as your reference point
  • "Use current position as origin" option — ensure consistent use session to session
  • Material not placed at same reference point — use physical stops or jig on machine table
  • Galvo zero offset drifting — recalibrate galvo offset in Laser Settings if shift is consistent
8. Curved Lines Not Smooth (Appear as Straight Segments)

Cause: Low arc precision setting in EzCad, or the source DXF file has low resolution curves.

Fix: In EzCad, Edit → Preferences → set "Curve accuracy" or "Arc approximation" to a smaller value (e.g., 0.001mm instead of 0.01mm). Also check the source design — if exported from Inkscape as DXF, ensure "Maximum deviation" is set very small (0.001mm) in the DXF export settings.

Software / Performance Issues

9. EzCad Crashes on Large Files

Causes:

  • Very complex paths (high node count) — simplify paths in Inkscape before exporting
  • Large bitmap images — fiber laser handles vectors better than bitmaps; convert photos to vectors or use EzCad's bitmap marking with reduced resolution
  • Low RAM PC — EzCad2 works fine on 2GB RAM for simple files but complex files need 4GB+

Fix: Simplify designs in Inkscape (Path → Simplify, Ctrl+L). Split very large jobs into sections.

10. F2/Enter Fires Laser Unexpectedly

Warning: F2 in EzCad immediately begins marking. If your workpiece is not in position, or you accidentally press F2, the laser fires at whatever is under the head. Always position your workpiece before pressing F2.

Safety tip: Set a "delay before mark" in marking settings (1–2 seconds) so you have time to confirm position. Use the footpedal or a confirmation dialog if your machine supports it.

11. Red Dot Pointer Misaligned with Actual Mark Position

Cause: The red dot pointer (low-power visible diode) is offset from the actual IR laser beam. This is normal — it's calibrated at purchase but can drift.

Fix: Recalibrate the red dot offset in Laser Settings. Mark a dot at known position, measure the offset between where red dot pointed and where mark appeared, enter the offset values in the Red Dot Offset fields (X, Y in mm). Your machine supplier can also do this calibration.

MOPA-Specific Issues

12. "Frequency Out of Range" Error (MOPA)

Cause: You've set a frequency outside the MOPA laser's valid range. Different MOPA lasers have different ranges — typically 1kHz to 4000kHz, but the specific range depends on the pulse width selected.

Fix: Check your laser's specification sheet for frequency range at each pulse width. At very short pulse widths (1–2ns), maximum frequency is limited. At long pulse widths (200ns+), minimum frequency is higher. Stay within the specified range for your pulse width.

13. Q-Pulse Width Not Changing Effect (Wrong Laser Type)

Cause: In EzCad Laser Settings, the "Laser Type" must match your actual laser — MOPA (JPT, Raycus MOPA) vs. Q-switch (standard Raycus, IPG). If set to Q-switch when you have MOPA, pulse width changes do nothing (Q-switch pulse width is fixed internally).

Fix: File → Laser Settings → Laser Type. Set to "MOPA" or the specific model (JPT-M6, Raycus MOPA, etc.) that matches your laser source. Confirm with your machine supplier.

14. Machine Marks but Galvo Speed Seems Capped

Cause: EzCad has a "Max Speed" limit in Laser Settings. Also, some controllers have firmware-level speed limits.

Fix: Check Laser Settings → Max Speed parameter. Increase if needed (up to the galvo scanner's rated maximum, typically 6000–8000 mm/s). Note: very high speeds reduce mark quality — above 2000mm/s marks become increasingly inconsistent on metal.

India-Specific Issues

15. Power Surge Damaged Controller Card

India context: Unstable grid power (voltage spikes, brown-outs) is the #1 cause of premature fiber laser controller failure in India.

Prevention:

  • Servo voltage stabilizer (servo AVR): ₹3,000–8,000 for a 3kVA unit — essential investment. Places like Luminous, V-Guard, Servokon.
  • Online UPS: Better than stabilizer alone — provides clean power and ride-through during brief cuts. ₹8,000–20,000 for suitable capacity.
  • Never run fiber laser directly from raw mains in areas with poor grid quality

If already damaged: Controller card replacement from JCZ India distributor. Cost: ₹8,000–25,000 for a replacement JCZ E-series card. Always have the machine serial number and card model ready.

16. Summer Heat Causing Controller Errors / Random Shutdowns

India context: In summer (April–June), ambient temperatures in workshops can exceed 40°C. Fiber laser controllers have thermal protection that triggers shutdown at high temperatures.

Fixes:

  • Keep machine room below 35°C — add a room AC or at minimum a dedicated wall-mount AC for the machine corner
  • Ensure at least 30cm clearance on all sides of machine for airflow
  • Check the controller cooling fan is running (small fan on controller card) — replace if failed
  • The fiber laser source itself has a water chiller or air cooler — ensure chiller water is clean and coolant level is adequate
  • Clean dust from cooling fins annually (compressed air)
17. Monsoon Humidity Causing Dongle / Connection Issues

India context: High humidity (80–95% RH in monsoon) can cause condensation on PCB surfaces and USB contacts.

Fixes:

  • Silica gel packets inside machine cabinet — replace or recharge every 3 months
  • Never leave IPA or cleaning solvents open near machine — raises humidity and creates explosion risk
  • If machine has been off overnight in monsoon: power on and let warm up for 15–20 minutes before marking — controller PCB needs to dry slightly
  • Dehumidifier in machine room if budget allows — target RH below 60%

Advanced Issues

18. EzCad3 License Errors

EzCad3 uses a cloud-based or file-based license instead of a physical dongle. Common errors:

  • "License expired": EzCad3 licenses may be time-limited — check renewal with your machine supplier
  • "License not valid for this machine": License is locked to PC hardware fingerprint. If you changed your PC, contact JCZ to re-register
  • "Cannot connect to license server": EzCad3 may require internet connection for license validation — ensure internet access or switch to offline license mode
19. Galvo Correction File Lost (Edge Distortion After Windows Reinstall)

Cause: After Windows reinstall or when setting up on new PC, the .cor (galvo correction) file is not automatically transferred.

Fix: The .cor file should be in the EzCad installation folder (typically C:\EzCad2\ or on the CD/USB that came with machine). Copy it to new installation. If lost: contact your machine supplier — they should have a copy of the calibration file for your machine serial number. Some suppliers re-do galvo calibration on-site (may cost ₹1,000–3,000 service call).

20. Jump TC Errors — Dots at Start of Lines

Cause: "Jump TC" (Jump Time Constant) is the delay EzCad waits for the galvo to settle at a new position before firing. If too short, laser fires while galvo is still moving — creates dots or smears at line starts.

Fix: File → Laser Settings → Jump TC. Increase value by 0.1ms increments (start from 0.3ms, test up to 1.5ms). Higher value = better quality but slower overall marking (more delay between objects). Also check "Laser On TC" and "Laser Off TC" parameters.

21. Mark Jumps to Wrong Position (Origin Drift)

Cause: The "current position" origin in EzCad is not saved between sessions. If machine is turned off/on or EzCad is restarted, the origin resets.

Fix: Always re-home your origin at the start of each session. Use physical reference stops on your machine table. If doing batch work across multiple sessions, use a jig plate with fixed pin positions for consistent material placement. PROJECT 4: Titanium Color Marking — Premium Stamp Plates

Business context: Color-marked titanium is the premium end of the stamp plate market. Indian craft buyers are increasingly willing to pay ₹2,000–5,000 for a beautiful colored mandala titanium stamp. MOPA only — standard Q-switch cannot do this.

Color Marking Process

1
Separate design by intended color zones
In Inkscape: each color zone on a separate layer → export as separate DXFs, or combine into one file with each zone as a separate EzCad pen.

Assign different pens for each color zone
Pen 1 = gold zone settings. Pen 2 = blue zone settings. Pen 3 = purple zone settings. (See color chart in Tab 12.)

Fiber Laser Maintenance Checklist

Monthly

  • Clean F-theta lens: Use proper lens tissue (never paper towel) + spectroscopy-grade IPA or acetone. Single-direction wipe. Dirty lens = inconsistent marks + potential damage
  • Check galvo mirrors: Visually inspect for dust, pits, or oxidation. Do not touch mirror surfaces. Report any damage to supplier immediately
  • Check fiber cable bend radius: Ensure the fiber delivery cable (from laser source to galvo head) has no tight bends (minimum 100mm radius). Bent fiber = degraded power
  • Check cooling system: Water chiller coolant level, chiller filter cleanliness. Air-cooled systems: clean dust from heatsink fins with compressed air
  • Check ventilation: Machine enclosure ventilation fan working, extraction exhaust not blocked
  • Clean work surface / Z-axis table: Remove metal dust, IPA residue with dry cloth. Metal dust on electrical connectors causes problems
  • Test mark quality: Run a standard test pattern on scrap SS304, compare to your reference image. Any degradation = investigate

Annually

  • Professional laser power output measurement (power meter) — compare to rated output
  • Galvo field recalibration if marks show increasing edge distortion
  • Water chiller coolant replacement

India-Specific Tips

Power Quality — Voltage Stabilizer

India's grid has significant voltage fluctuation (170–250V in many areas). The fiber laser controller is sensitive to voltage spikes. A servo voltage stabilizer (₹3,000–8,000) is the single most important investment after buying the machine. Brands: Servokon, V-Guard, Luminous.

Summer Temperature Control

Keep machine room below 35°C. A window AC unit (1.5 ton) in the machine area is worth the investment. Signs of overheating: random shutdowns, "thermal protection" errors, inconsistent laser output.

Monsoon Humidity

Use silica gel packets in machine cabinet. Never leave IPA/acetone containers open. Allow machine to warm up 15 minutes after being off overnight during monsoon before marking.

Power Cut During Marking

EzCad does not auto-resume after power cut. The job is lost. Check if design is partially marked — if so, you may need to restart the entire job. With a UPS, brief cuts (1–5 min) are handled without job loss.

Dongle Backup — JCZ Registration

Register your dongle serial number with JCZ (or your India supplier) immediately after purchase. In India, dongle replacement typically takes 1–3 weeks. Keep the dongle serial number photographed and stored in cloud. Never keep the dongle attached when machine is not in use — reduces physical damage risk.
