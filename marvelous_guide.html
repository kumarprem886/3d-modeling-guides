2D Pattern Tools

Rectangle tool — draw fabric panel rectangles R
Polygon tool — draw custom panel shapes P
Move anchor points on panel edges Edit Curve Point tool
Add fold lines / seam lines inside panels Internal Line tool
Connect edges of two panels (sewn in simulation) Add Seam
Sew specific segment to segment (length-matched) Segment Sew
Sew in any direction for complex garments Free Sew
Stitch pattern for realistic seam appearance M-Stitch
Tip: Plan your panels in 2D like a real dressmaker pattern before simulating. Accurate 2D shapes produce much better 3D cloth results — don't rush to simulate.

Simulation Controls

Simulate — drape cloth onto avatar Space
Simulation quality — lower = higher quality Particle Distance setting
Reset garment to arrangement positions Ctrl + Shift + R
Pin cloth in current simulated state Freeze
Add wind force to simulation Simulation → Wind
Add air pressure (for inflatable shapes) Pressure setting
Adjust gravity strength Gravity setting
Load avatar or OBJ character mesh File → Load Avatar
Position panels around avatar before simulating Arrangement points
Tip: Lower Particle Distance (5–10mm) gives very high quality cloth but is much slower. Use 20mm for quick tests, then drop to 5mm for your final simulation pass.

Fabric Properties

Preset fabrics: Cotton, Silk, Denim, Leather, Knit Fabric Library
Scale fabric in warp/weft directions Shrinkage setting
Controls stiff vs droopy cloth behavior Bending stiffness
High = grips itself, low = slides freely Friction setting
Physical thickness of the fabric layer Thickness setting
Assign color or texture image to fabric Color / Texture slot
Edit all fabric settings for selected panel Select panel → Property Editor (right panel)
Tip: Use the Fabric Library presets as starting points, then adjust Bending stiffness for your specific look. Denim is much stiffer than Silk — the presets give you the right ballpark immediately.

Export & 3D Pipeline Integration

Export garment as OBJ (with or without UV) File → Export → OBJ
Export with quad mesh and unified UVs File → Export → OBJ with Unified UV Coordinates
Increase polygon count for smoother look on export Subdivision on export setting
Export multiple clothing states / poses File → Export → Multiple Poses
Import into Blender as cloth or static mesh Blender → OBJ Import
Import into ZBrush for surface detail sculpting ZBrush → Import OBJ
Export animated cloth cache for Maya / Houdini / Blender File → Export → Alembic (.abc)
Bake wrinkles from high-sim to game-res mesh Normal map bake in Marmoset Baker
Add fabric texture, print, embroidery Substance Painter → Import OBJ
Pipeline tip: The standard workflow is: Marvelous Designer (cloth sim) → Marmoset Baker (bake wrinkle normals) → Substance Painter (texture) → Marmoset Toolbag (final render). Export OBJ with Unified UVs at each handoff.
Alembic tip: Use Alembic (.abc) export to carry animated cloth simulations into Blender or Houdini as a cache — no re-simulation needed in the target app.
