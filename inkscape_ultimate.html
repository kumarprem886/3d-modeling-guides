Getting Started with Inkscape
Free, open-source vector graphics editor — the industry standard for SVG work
What is Inkscape? Inkscape is a professional vector graphics editor that works natively with SVG files. Unlike raster editors (Photoshop), everything you draw stays infinitely scalable — perfect for laser engraving, stamp plates, logos, and technical illustrations.
📥 Installation
  • Download from inkscape.org — always get the latest stable
  • Windows: use the .msi installer (not the portable zip)
  • Current stable: Inkscape 1.3.x
  • Enable "Associate SVG files" during install
  • Optional: install Python 3 for extensions to work
📐 First Document Setup
  • Ctrl+Shift+D → Document Properties
  • Set units to mm for physical work
  • Enter your plate/canvas dimensions in mm
  • Set display DPI to 96 (SVG default)
  • Check "Scale strokes with object" → ON
🖥 Interface Overview
Toolbox (left)

All drawing tools — selector, node editor, bezier pen, shapes, text, zoom. Press the keyboard shortcut letter to activate any tool instantly.

Tool Controls Bar (top)

Context-sensitive options for the active tool. Changes when you switch tools — width, height, corner radius, node types, etc.

Canvas

The white rectangle is your document page. Objects outside the page won't appear in most exports. Middle-click drag to pan; scroll to zoom.

Fill & Stroke Dialog

Shift+Ctrl+F — controls color, opacity, gradients, stroke width, and dash patterns.

XML Editor

Ctrl+Shift+X — raw SVG attribute editor. Use it to manually set exact values or fix imported files.

Objects / Layers Panel

Object → Objects (Ctrl+L for Layers). Shows the full object tree. Drag to reorder, click eye/lock icons.

Status Bar (bottom)

Shows cursor coordinates, selected object type, fill/stroke swatches, and zoom level. Critical for precision work.

Palette (bottom)

Quick fill color — left-click to set fill, right-click to set stroke. Shift+click to add to stroke. Click the ✕ to remove fill.

🎯 Core Concepts
Vector vs Raster: Inkscape works with vectors — mathematical descriptions of shapes. No pixels. You can scale to any size with zero quality loss. This is why SVG is the format of choice for laser engravers and stamp plates.
Objects

Rectangles, circles, paths, text, groups — everything on canvas. Each object has fill, stroke, position, size, and transform attributes.

Paths

The most powerful object type — a series of nodes connected by straight or curved segments. All shapes can be converted to paths.

Groups

Multiple objects treated as one. Press Ctrl+G to group, Ctrl+Shift+G to ungroup. Double-click to enter a group.

Layers

Like Photoshop layers. Organise artwork into named layers. Lock layers to prevent accidental edits. Ctrl+L to open.

Fill vs Stroke

Fill = the inside color. Stroke = the outline. For laser engraving, you usually want black fill, no stroke — or stroke-only for cutting.

Viewbox

The internal coordinate system of an SVG. Inkscape manages this automatically, but understanding it helps when editing SVGs in code or the XML editor.

⚙ Recommended Preferences

Go to Edit → Preferences (Ctrl+Shift+P) and set these:

  • Input/Output → SVG output: "Plain SVG" by default
  • Snapping: Enable "Snap to nodes, paths, and handles"
  • Rendering: Set display color profile to sRGB
  • Tools → Node: Show handles for selected nodes always
  • Behavior → Clones: "Select original" on clone select
  • Input → Scrolling: Zoom with scroll wheel (not pan)
  • Interface → Zoom: "Zoom to fit drawing" on open
  • Save → Autosave: Every 5 minutes, keep 5 backups
Tools Reference
Every tool in the toolbox — what it does, keyboard shortcut, and pro tips
Selection & Transform
↖ Selector S / F1

Select, move, scale, rotate objects. Click once to select, click again to toggle rotate/skew handles. Drag a rubber-band to select multiple.

◦ Node Editor N / F2

Edit individual nodes and bezier handles on paths. Tab to cycle nodes. Del to remove a node. Insert to add node on segment.

⊞ Transform Shift+Ctrl+M

Precise move, scale, rotate, skew by exact amount. Essential for aligning objects to exact mm positions.

✕ Tweak W / F7

Warp, push, attract, repel, roughen nodes. Useful for organic distortion effects on paths.

Drawing Tools
✏ Bezier / Pen B / P

Draw straight segments (click) or curves (click+drag for handles). Enter to finish. Escape to cancel. Most important drawing tool.

✒ Pencil (Freehand) P (mode)

Draw freehand paths. Smoothing slider controls how tightly it follows your mouse. Higher smoothing = fewer nodes.

🖊 Calligraphy C / F6

Variable-width strokes. Use with a drawing tablet for natural calligraphy or brush lettering effects.

🪣 Paint Bucket U / F8

Flood-fill enclosed areas, creating a new filled path. Threshold controls how precisely it detects edges.

✏ Spray A

Scatter copies of selected objects across the canvas. Great for decorative repeat patterns.

⬛ Rectangle R / F4

Draw rectangles. Hold Ctrl for perfect squares. Rx/Ry in toolbar for rounded corners.

⬤ Ellipse / Circle E / F5

Draw ellipses. Hold Ctrl for perfect circles. Start/End angles for arcs and pie slices.

★ Star / Polygon * / F9

Draw stars and regular polygons. Set corners, spoke ratio, and rounded/random parameters in toolbar.

〰 3D Box X

Draw a box in perspective using vanishing points. Isometric box drawing with controlled perspective.

〰 Spiral I / F9

Draw Archimedean or logarithmic spirals. Set turns, inner radius, and divergence.

T Text T / F8

Click for single-line text, click+drag for text box (auto-wraps). Font, size, kerning, leading in toolbar.

📏 Measure M

Click and drag to measure distance and angle between two points. Shows in document units.

View & Navigation
🔍 Zoom Z / F5

Click to zoom in, right-click to zoom out, or drag a rectangle to zoom to that area. Use scroll wheel from any tool.

✋ Pan H / middle-drag

Pan the canvas. Also works with middle-mouse drag from any tool — you don't need to switch to pan tool.

💧 Dropper F7 / Dropper

Pick a color from anywhere on canvas and assign it to the selected object's fill or stroke.

⬚ Pages Page tool

Inkscape 1.2+ supports multiple pages in one document. Add, resize, and reorder pages for multi-plate work.

Pro tip: You almost never need to click the toolbox. Learn the keyboard shortcuts — S selector, N nodes, B bezier, R rect, E ellipse, T text — and you'll work 3× faster.

Paths & Nodes
The heart of vector work — mastering paths unlocks everything
The golden rule: For laser engraving and stamp plates, all artwork must be paths — not shapes, not text. Run "Object to Path" on everything before exporting.
📍 Node Types
  • Cusp (Corner) — handles move independently, creates sharp corners
  • Smooth — handles stay in a line, creates smooth curves
  • Symmetric — handles equal length, perfectly smooth S-curves
  • Auto-smooth — Inkscape picks the best smooth direction automatically
Change node type: Select node in Node tool, then use the type buttons in the toolbar, or press Shift+S, Shift+Y, Shift+U.
✂ Segment Types
  • Line segment — straight line between two nodes. Press Shift+L
  • Curve segment — bezier curve with handles. Press Shift+U
  • Select a segment (click between nodes) and switch its type in toolbar
  • Mix straight and curved segments freely in one path
🔀 Boolean Path Operations

Booleans combine two paths into one. Select two paths, then use Path menu or keyboard shortcut. Bottom object = first, top = second.

🖊 Drawing with Bezier Pen
  1. Press B to activate Bezier pen
  2. Click to place a straight corner node
  3. Click + drag to place a curve node with handles
  4. Continue clicking to add more nodes
  5. Click the first node or press Enter to close/finish
  6. Press Esc to cancel and discard
  7. Hold Ctrl while placing to snap to 15° angle increments
🔧 Node Editing Tips
  1. Press N to enter Node editor
  2. Click a node to select it, shift+click to add to selection
  3. Tab / Shift+Tab to cycle through nodes
  4. Drag a node to move it
  5. Drag a handle to adjust curve tension/direction
  6. Del to remove a node (path reconnects)
  7. Ins to insert a node on a segment
  8. Drag a segment directly to bend it
⚙ Path Effects (LPE)

Live Path Effects transform a path non-destructively. Find them at Path → Path Effects (Ctrl+&).

Roughen

Adds random noise to a path — useful for hand-drawn, organic look on stamp designs.

Offset

Expand or shrink a path by a fixed distance. Better than stroke-to-path for insets/outsets.

Interpolate Sub-paths

Morphs between two shapes smoothly — great for creating transition effects.

Pattern Along Path

Tiles a shape along a curved path — borders, rope effects, decorative frames.

Envelope Distort

Warp a path inside a bounding envelope — perspective distortion, arc text.

Corners

Round, chamfer, or inverse-round individual corners with precise control.

Warning: LPEs are live — the original path is preserved. When exporting for laser/stamp, flatten the LPE first: Path → Flatten Beziers or save as Plain SVG which auto-flattens many effects.
🔎 Simplify & Clean Up Paths

Simplify (Ctrl+L)

Removes redundant nodes while keeping the overall shape. Press repeatedly to simplify further. Essential after tracing bitmaps.

  • Reduces file size
  • Smoother laser cutting
  • Faster machine movement

Clean Up Document

File → Clean Up Document — removes unused defs, styles, and empty groups. Run before exporting final SVG.

  • Removes orphaned gradients
  • Removes unused patterns
  • Reduces file size significantly
Fill & Stroke
Control how objects look — color, gradients, patterns, and outlines
For laser/stamp work: Use solid black fill, no stroke (or stroke-only, no fill for cut lines). Avoid gradients, patterns, and transparency — laser engravers read solid fills, not gradients.
🎨 Fill Types
  • None (X) — invisible fill, object is transparent inside
  • Flat color — solid single color. Use hex codes for precision
  • Linear gradient — smooth transition along a line
  • Radial gradient — smooth transition from a center point outward
  • Pattern — tile a pattern from the pattern library or custom
  • Swatch — fill from a named palette color
  • Unset paint — inherits fill from parent group
✏ Stroke Types
  • Stroke paint: same options as fill (none, flat, gradient)
  • Stroke width: set in px, mm, or pt
  • Dash pattern: predefined dashes or custom dash/gap lengths
  • Line cap: Butt (flat end), Round, Square
  • Join: Miter (pointy), Round, Bevel (flat corner)
  • Markers: Arrowheads, dots at path start/end/mid
🌈 Working with Colors

Color Models

  • RGB — screen colors (0–255 per channel)
  • HSL — hue, saturation, lightness — intuitive for adjusting
  • CMYK — print colors (0–100% per channel)
  • Hex — e.g. #1a2b3c — paste directly into the hex box
  • HSV — hue, saturation, value — similar to Photoshop color picker

Color Picker Tips

  • Press F7 or select Dropper tool to pick from canvas
  • Hold Shift to assign picked color to stroke instead of fill
  • The dropper picks the rendered color including gamma correction
  • Use the RGBA hex box at the bottom of Fill&Stroke to paste hex codes
  • Inkscape supports 8-digit hex (RGBA) — last 2 digits = alpha
🖊 Stroke to Path

Ctrl+Alt+C — converts a stroked path into a filled shape. The stroke outline becomes the new path boundary.

When to use this: If you have a logo where the "outline" is a thick stroke, convert to path before laser engraving. Otherwise the laser only traces the center line, not the visible outline width.
  • After conversion, the object has fill but no stroke
  • Original stroke width becomes the width of the new filled shape
  • Good for ensuring consistent visual output across different software
  • Required for most laser cutter file formats (DXF, etc.)

Text & Typography
From basic labels to complex curved text — then converting to paths for production
Always convert text to paths before export! Other software (laser cutters, CNC, Plate Composer) cannot render Inkscape fonts. Path → Object to Path (Shift+Ctrl+C) converts text to outlines permanently.
✍ Creating Text
  1. Press T to activate Text tool
  2. Click anywhere for single-line text (grows as you type)
  3. Click + drag to create a text box (text wraps inside)
  4. Type your text. Use Enter for new lines
  5. Select all text with Ctrl+A inside the text object
  6. Set font, size, style, alignment in the toolbar
  7. Press Esc or S to exit text editing and select the object
🔤 Text Formatting
  • Font: Set in toolbar or Ctrl+Shift+F (Text & Font dialog)
  • Size: In pt, px, mm — set in toolbar top
  • Kerning (letter spacing): Alt+left/right arrow to adjust between characters
  • Leading (line spacing): Ctrl+Alt+up/down arrow
  • Bold / Italic: Toolbar buttons or font family selection
  • Align: Left, Center, Right, Justify in toolbar
  • Superscript / Sub: Format menu inside text tool
〰 Text on a Path
1
Draw the path

Draw a circle, arc, or any path with the shape/bezier tools. This is the guide path — it won't appear in the final output.

2
Create text

Use the Text tool (T) to type your text somewhere on the canvas. It doesn't need to be near the path yet.

3
Select both, put on path

Select the text object first, then Shift+click the path. Go to Text → Put on Path. The text flows along the path.

4
Adjust offset

Select just the text, use Alt+left/right to slide along the path. Or set the offset in Text & Font dialog.

5
Remove the path fill/stroke

Select the guide path, set fill and stroke to none so it's invisible in the export. The text stays on it.

6
Convert to path for export

Select text → Shift+Ctrl+C (Object to Path). Unlink from guide path. Text is now a plain path — safe to export.

🅰 Flowed Text

When you click+drag with Text tool, you create a "flow text" object. Text wraps inside the rectangle.

Warning: Flowed text doesn't export correctly in all SVG viewers. Before export, convert: Text → Convert to Text, then Object to Path.
🔠 Text to Paths Workflow
  1. Finish all text editing
  2. Select all text objects (Edit → Find → type: text)
  3. Shift+Ctrl+C → Object to Path
  4. Text is now outline paths — uneditable but portable
  5. Run Ctrl+Shift+G to ungroup if needed
Stamp plate text tips: For raised letter stamps, your text fill should be black. The stamp will push the black areas into the material. If you want inset letters (debossed), invert with a Difference boolean — subtract the text from a solid rectangle.
Export & Save
Get your artwork out of Inkscape in the right format for every use case
📁 Save vs Export — Key Difference

Save (Ctrl+S)

Saves as Inkscape SVG — includes Inkscape-specific metadata, layer info, LPE data. Use this for your working file.

  • Preserves all Inkscape features
  • File is larger (has extra metadata)
  • Not ideal for sharing with other software

Save a Copy → Plain SVG

Strips Inkscape-specific tags and produces a clean, standard SVG. Use this for sharing, laser cutters, and the Plate Composer.

  • No Inkscape namespaces
  • Smaller file size
  • Works in all SVG viewers
  • Flattens some LPEs
🖼 Export PNG (Ctrl+Shift+E)

Export Options

  • Document — exports the full page at set DPI
  • Page — same as document in most cases
  • Drawing — exports only the bounding box of all objects
  • Selection — exports only the selected objects
  • Custom — set exact px dimensions and position

DPI Guide

  • 96 DPI — screen display (SVG default)
  • 150 DPI — draft print quality
  • 300 DPI — standard print quality
  • 600 DPI — chemical etch transparency film
  • 1200 DPI — high-precision film / PCB work
📤 File Formats
SVG (Plain)

Best for: Plate Composer, web, most laser software. Standard, clean, no Inkscape metadata.

PNG

Best for: Chemical etch film (600 DPI), previews, sharing. Raster — not scalable.

DXF (R14)

Best for: EzCad (fiber laser), CNC machines. Extensions → Render → DXF. Use R14 format.

PDF

Best for: Print shops, Lightburn import. Preserves vector. File → Save As → PDF.

EPS

Best for: Older professional print workflows. Save As → EPS. Requires Ghostscript installed.

EMF / WMF

Best for: Microsoft Office embedding. Extensions → Render → EMF or WMF.

AI (Adobe Illustrator)

Save as PDF — AI can open it. True AI format requires Adobe software.

Inkscape SVG

Working file format only — not for sharing or production. Has inkscape: namespace that causes errors in other tools.

🔖 Export Checklist (Before Any Export)
1
Convert all text to paths

Select all text → Shift+Ctrl+C. Without this, exported SVG shows wrong or missing fonts in other apps.

2
Object to Path on all shapes

Rectangles, circles, stars → select all → Shift+Ctrl+C. Ensures compatibility with all readers.

3
Flatten LPEs

Path → Flatten Beziers, or simply save as Plain SVG which auto-flattens most live effects.

4
Set all fills to black, remove strokes

For laser/stamp: Edit → Find/Replace → Select by fill, set to #000000. Remove stroke (right-click palette → remove stroke).

5
Clean Up Document

File → Clean Up Document — removes unused defs, gradients, and patterns. Keeps file size minimal.

6
Save as Plain SVG

File → Save a Copy → Plain SVG. This is the clean file to hand off to laser software or the Plate Composer.

Stamp Plate Workflow
Complete production pipeline — from raw SVGs to laser-ready plate
Goal: Arrange multiple clean SVG designs on a plate-sized canvas (e.g. 60×90mm), export as one clean SVG or PNG ready for your laser engraver or chemical etching process.
📐 Step 1 — Set Up the Document
1
Create new document at plate size

Ctrl+Shift+D → Document Properties. Set units to mm. Enter plate dimensions — e.g. Width: 60mm, Height: 90mm.

2
Set up a grid for alignment

Document Properties → Grids tab → Add Rectangular grid. Set spacing to 1mm. Enable "Snap to grid" (# to toggle).

3
Draw the plate border

Use Rectangle tool to draw a rectangle the exact plate size. Set stroke to red 0.1mm, fill to none. This is your visual boundary — delete before export.

🧹 Step 2 — Clean Each Design SVG
1
Open each SVG and ungroup everything

Select all (Ctrl+A) → Ungroup (Ctrl+Shift+G) repeatedly until nothing is grouped. Remove all metadata, layers, and guide paths.

2
Object to Path on everything

Select all → Shift+Ctrl+C. Converts text, shapes, and effects to plain paths.

3
Remove white fills and backgrounds

Delete any full-size background rectangles. Select all white objects (Edit → Find/Replace by fill) and delete or set fill to none.

4
Set all remaining fills to black

Select all → open Fill & Stroke → set fill to Flat color → enter #000000. Set stroke to none.

5
Save as Plain SVG

File → Save a Copy → Plain SVG. Use a clean filename. This is the file you'll import into the plate layout.

📋 Step 3 — Arrange on the Plate
1
Import each clean SVG

Ctrl+I → select SVG → choose Link: No, Embed. The design appears on canvas. Or use the Plate Composer web tool to batch-arrange all designs automatically.

2
Resize each design uniformly

Select the design group. In W/H fields in toolbar, enter the target size. Lock the aspect ratio icon (chain icon) must be active. Type new width → Tab → height auto-updates.

Ctrl+drag corner to scale proportionally
3
Align and distribute evenly

Select all designs → Shift+Ctrl+A → Align & Distribute. Use "Make horizontal/vertical gaps between objects equal" to space them evenly with consistent margins.

4
Check no design overlaps plate edge

Zoom out (5). Ensure all designs are within the plate boundary. Use Document Properties to enable "clip to page" if needed.

Faster alternative: Use the Plate Composer tool — load your folder of SVGs, click "✨ Suggest" for optimal settings, then Compose. It auto-arranges all designs and lets you resize/move interactively. Export directly as laser-ready SVG or 600 DPI PNG.
⚙ Step 4 — Export

For Laser Engraving (Lightburn / xTool)

  1. Delete the red border rectangle
  2. File → Save a Copy → Plain SVG
  3. Import into Lightburn / XCS
  4. Set fill layer to engrave at desired speed/power
  5. Frame → Engrave

For Chemical Etching (Film Print)

  1. Ensure all fills are solid black, no grays
  2. Export PNG at 600 DPI (Ctrl+Shift+E)
  3. Print on transparency film at 100% size
  4. Apply to photoresist-coated plate
  5. UV expose → develop → etch → strip resist

Advanced Techniques
Clones, patterns, extensions, XML editor, and power-user workflows
🔗 Clones

A clone is a linked copy of an object. Edit the original → all clones update automatically.

  • Alt+D — create a clone of selected object
  • Alt+Shift+D — select the original of a clone
  • Clones can have their own position, scale, rotation
  • Edit → Clone → Unlink Clone to make independent
  • Tiles of Clones: Edit → Clone → Create Tiled Clones — powerful repeat pattern tool
🪟 Symbols

Object → Symbols — a library of reusable symbols. Import symbol collections or create your own.

  • Inkscape ships with SVG standards symbols
  • Place a symbol → use element in SVG (like a smart clone)
  • For stamp plates: create each design as a symbol, place multiple times
  • Convert to path before export (symbols don't export cleanly in all software)
🔁 Create Tiled Clones

Edit → Clone → Create Tiled Clones — the most powerful repeat tool in Inkscape.

What you can do

  • Create grids, circles, rows of clones
  • Vary color, opacity, scale, rotation across the tile
  • Set rows, columns, spacing
  • Symmetry options: P1, P2, P3, P4, P6, mirror variants
  • Use it for decorative borders on stamp plates

Workflow

  1. Select the object to tile
  2. Edit → Clone → Create Tiled Clones
  3. Set rows × columns, spacing in mm
  4. Apply — a grid of clones appears
  5. Edit the original to update all clones
  6. Unlink and flatten before export
⚙ XML Editor

Ctrl+Shift+X — direct access to the raw SVG XML of any selected object.

What you can do

  • Set any attribute to an exact value
  • Add custom attributes (data-*, aria-*)
  • Fix imported SVGs with broken attributes
  • Set viewBox precisely
  • Remove unwanted Inkscape namespace tags
  • Debug path data (d attribute)

Common Uses

  • Set width / height to exact mm value
  • Remove sodipodi:* and inkscape:* attrs
  • Add id attribute for laser layer naming
  • Fix viewBox for mis-sized imports
  • Set fill-rule="evenodd" for complex paths
🧩 Extensions

Extensions menu — Inkscape ships with dozens of useful extensions. More at inkscape.org/gallery/=extension/.

Render → Grid

Generates a precise grid overlay on canvas. Useful for planning stamp plate layouts.

Generate from Path → DXF

Exports selected paths to DXF format compatible with EzCad and CNC routers.

Gcodetools

Converts SVG paths to G-code for CNC mills. Set depth, speed, and tool diameter.

Modify Path → Flatten Beziers

Converts curves to short line segments — required for some CNC/laser software that can't handle beziers.

Color → Palette

Replace all colors with the nearest palette color. Useful for standardising designs to two-color (black + white).

Text → Remove Kerning

Strips kerning data from text before Object to Path — prevents spacing issues in some software.

Render → Barcode

Generate QR codes, barcodes, and other machine-readable patterns as clean SVG paths.

Jessy Ink

Presentation mode — step through a document layer by layer like a slideshow.

Bitmap Trace (Potrace)

Converts raster images to vector paths. Edit → Trace Bitmap (Shift+Alt+B).

🖼 Trace Bitmap (Vectorise Images)

Path → Trace Bitmap (Shift+Alt+B) — convert raster images to vector paths using Potrace.

Modes

  • Brightness cutoff — traces based on lightness threshold. Best for high-contrast logos.
  • Edge detection — traces outlines of objects. Good for complex images.
  • Color quantization — traces each color as a separate path layer. For multi-color art.
  • Autotrace — uses autotrace engine (if installed). More options but slower.

Tips for Best Results

  • Pre-process in an image editor: increase contrast, convert to pure B&W
  • Higher threshold = more of the image is black
  • Simplify after tracing: Ctrl+L several times
  • Remove small noise paths: Edit → Find → by size → delete tiny paths
  • Check result at 100% zoom before using for engraving
  • For stamps: trace at 2–3× final size then scale down (more detail retained)
📏 Guides & Snapping

Guides

  • Drag from ruler to create a guide line
  • Double-click a guide to set exact position in mm
  • Drag guide back to ruler to delete it
  • Edit → Guides — toggle visibility
  • File → Document Properties → Snap tab

Snapping (% toggle)

  • % — toggle snapping on/off globally
  • Snap to grid (1mm grid is ideal for stamps)
  • Snap to nodes and handles
  • Snap to bounding box edges and corners
  • Snap to page edge — keeps designs inside plate
🚀 Power User Tips
Batch export with command line: inkscape --export-type=png --export-dpi=600 --export-filename=out.png input.svg — export any SVG to 600 DPI PNG without opening the GUI.
Batch clean SVGs: Use Inkscape's command line to run plain SVG export on all files: inkscape --export-plain-svg=clean.svg input.svg
Align to page: In Align & Distribute, set "Relative to" to Page — then Center on Vertical Axis and Center on Horizontal Axis will center your object on the page perfectly.
Object size in mm: With Selector tool active, W and H boxes in the toolbar show size. Make sure the units dropdown shows mm, not px. Change document units in Document Properties → Display tab.
Multiple pages (1.2+): Inkscape 1.2+ supports multiple pages. Use the Pages tool to add pages — great for managing multiple stamp plate sizes in one working file.
