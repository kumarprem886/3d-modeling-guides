Import & Basic Repair

  • Open meshFile → Import Mesh
  • Geometric measuresFilters → Quality Measure → Compute Geometric Measures
  • Check holes / topologyFilters → Mesh Layer → Compute Topological Measures
  • Fill holesFilters → Remeshing → Close Holes
  • Remove duplicate verticesFilters → Cleaning → Remove Duplicate Vertices
  • Remove duplicate facesFilters → Cleaning → Remove Duplicate Faces
  • Remove zero-area facesFilters → Cleaning → Remove Zero-Area Faces
  • Remove unreferenced vertsFilters → Cleaning → Remove Unreferenced Vertices
  • Fix normalsFilters → Normals → Reorient All Faces Coherently
Tip
Run all cleaning filters (duplicates, zero-area, unreferenced) before any other operation — they fix most basic import issues.
Warning
After fixing normals, verify the result looks correct in solid view — reorientation can occasionally flip inside-out on complex meshes.

Mesh Processing

  • Reduce polygon countFilters → Remeshing → Quadric Edge Collapse Decimation
  • Suggested reduction50–70% with Preserve Normal checked
  • Subdivide meshFilters → Remeshing → Subdivision Surfaces
  • Smooth meshFilters → Smoothing → Laplacian Smooth
  • Smooth (preserve volume)Filters → Smoothing → Taubin Smooth
  • Scale / transformFilters → Normals → Transform: Scale
  • Merge multiple meshesFile → Import each, then Filters → Mesh Layer → Flatten Visible Layers
Tip
Use Taubin Smooth instead of plain Laplacian Smooth — it reduces noise while better preserving overall volume.
Info
Quadric Edge Collapse Decimation: check 'Preserve Normal' and 'Preserve Boundary' for best results.

Workflow — Repair for 3D Printing

  1. Import STL with potential issues (File → Import Mesh).
  2. Filters → Cleaning → Remove Duplicate Vertices, Faces, and Zero-Area Faces.
  3. Filters → Normals → Reorient All Faces Coherently (fix flipped normals).
  4. Filters → Remeshing → Close Holes (fill any gaps in the mesh).
  5. Check result: Filters → Quality Measure → Compute Topological Measures — should show 0 holes, 1 component.
  6. If too many polygons: Filters → Remeshing → Quadric Edge Collapse Decimation (50–70% reduction with Preserve Normal checked).
  7. File → Export Mesh As → STL → import into your slicer.
Tip
Step 5's Topological Measures check is your quality gate — only export when it shows 0 holes and 1 connected component.
Info
MeshLab is non-destructive for inspection — you can always undo or re-import if a filter produces unexpected results.
