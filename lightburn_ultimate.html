What is Lightburn?

Lightburn is an all-in-one laser control + design software that runs natively on Windows, macOS, and Linux. It replaces the fragmented workflow of using separate design tools and machine controllers — you design, configure cut settings, and fire the laser all from one application.

The key distinction Lightburn is your primary tool for sending jobs to the laser. It does basic vector design well. For complex artistic work, you'll still use Inkscape or Illustrator — but Lightburn handles everything from import to firing.

Key Panels Explained

Cuts / Layers Panel Most Important

Every object on the canvas is assigned to a color layer. Each layer has its own:

  • Mode: Line, Fill, Fill+Line, Offset Fill — controls how the laser moves
  • Speed: mm/s — how fast the laser head moves
  • Max Power / Min Power: % — laser power (min is for slow/acceleration zones)
  • Passes: how many times the laser runs each path
  • Priority / Order: drag layers to change execution order (bottom = first)
Pro tip Colors are just layer identifiers. Assign red = cut, black = engrave, blue = mark — whatever system makes sense for your workflow. The actual laser settings live in the layer, not the color.

Laser Panel

  • Start: sends the current job to the laser and begins cutting
  • Pause/Resume: pause mid-job (useful for checking settings)
  • Stop: emergency halt
  • Frame: traces the bounding box of your job with the laser off — essential for alignment
  • Home: sends laser to home position (0,0)
  • Go to Origin: moves to the set start point
  • Start from: Absolute Coords / User Origin / Current Position

Console Panel

Shows raw GRBL/controller commands and responses. Critical for debugging connection issues. If your laser isn't responding, check here for error messages like error:9 (no homing) or ALARM:1 (limit switch tripped).

Move Panel

Manual jog controls. Set step size (0.1mm to 100mm), then click directional arrows. Use this to position the laser head before starting a job, or to test machine movement.

Shape Properties Panel

When an object is selected, shows exact X/Y position, width, height, rotation angle. You can type exact values here for precision placement. Lock aspect ratio with the padlock icon.

Importing Files

Method 1: File > Import

Menu: File > Import (or Ctrl+I). Navigate to your file, click Open. Lightburn will ask about import settings for SVG/DXF.

Method 2: Drag & Drop

Simply drag any supported file from File Explorer onto the Lightburn canvas. This is the fastest method for quick imports. The file drops at the cursor position.

SVG Import — Getting It Right

SVG is the most common format for laser work. Most issues come from how the SVG was exported from Inkscape/Illustrator.

Set units to mm in Inkscape BEFORE exporting File > Document Properties > set units to "mm" and set the document size to your desired output size. If you export with px units, Lightburn may import at the wrong size (96dpi vs 90dpi mismatch).

Common SVG Import Issues & Fixes

Wrong import size SVG exported in pixels, not mm In Inkscape: File > Document Properties > change to mm, then re-export

Missing fills / shapes appear empty Shapes have no stroke, only fill — Lightburn uses strokes for cut lines In Inkscape: give shapes a stroke color (0.1mm width), remove fill for cut lines
Groups won't separate Objects are grouped in SVG Select all, press Ctrl+U repeatedly until fully ungrouped

Organizing with Layers

Assign different elements to different color layers for precise control over what gets cut vs engraved:

  • Black layer (00): Main engrave / vector mark
  • Red layer (01): Cut outline
  • Blue layer (02): Score / light mark
  • Assign by selecting objects, then clicking the color in the Cuts/Layers panel

SVG Import Tips for Stamp Plates

  • Set stroke width to 0.1mm (hairline) in Inkscape — thinner strokes = sharper laser lines
  • Ensure all paths are black stroke, no fill for clean vector engraving
  • Ungroup everything before exporting (Ctrl+U in Inkscape) so Lightburn can assign layers independently
  • Convert all text to paths in Inkscape: Text > Object to Path — avoids font dependency issues
  • Use Save as Plain SVG (not Inkscape SVG) to avoid metadata Lightburn might misread
  • Check your SVG in a browser first — if it looks wrong in Chrome, it'll import wrong in Lightburn

The Four Cut Modes

Line Mode Stamp Plates

The laser follows path outlines exactly — like drawing with a pen. Used for:

  • Vector engraving on metal (stamp plates)
  • Cutting through material
  • Scoring / folding lines
  • Engraving crisp outlines

Speed, power, and passes control depth. Multiple passes = deeper marks.

Fill Mode

The laser scans back and forth like a printer, filling the interior of shapes. Used for:

  • Engraving logos on wood, aluminum
  • Photo engraving
  • Filling text areas
  • Deep pocket engraving

Line interval controls density (DPI). Smaller interval = more passes = higher DPI.

Fill + Line Mode

Fills the shape first, then runs the outline — best of both worlds. Used for:

  • High-quality text engraving
  • Logos requiring crisp edges
  • Any engraving where edge quality matters

Takes longer but looks significantly better. Recommended for final production work.

Offset Fill Mode

Fills shapes with concentric inward rings instead of parallel lines. Used for:

  • Oval/round shapes (fewer direction changes)
  • Artistic engraving effects
  • Situations where linear fill creates banding

Slower to compute but can look more natural on organic shapes.

Min Power vs Max Power Explained

This confuses many beginners. On Ruida controllers and fiber lasers:

  • Max Power: power at full speed — used for most of the path
  • Min Power: power when the machine is slowing down at corners or reversing direction (acceleration/deceleration zones)
  • If Min Power is too low, corners will be underburned compared to straight sections
  • If Min Power = Max Power, every point on the path gets the same power — good for consistency but can over-burn corners
  • Rule of thumb: set Min Power = Max Power − 5 to 10%

Frequency Setting (Fiber Lasers)

  • Frequency = pulses per second (kHz) the laser fires
  • Lower frequency (20–30 kHz): more energy per pulse, deeper penetration, darker mark on SS304
  • Higher frequency (50–100 kHz): less energy per pulse, lighter surface mark, better for anodized aluminum
  • For SS304 stamp plates: 20–35 kHz recommended
  • MOPA lasers also have pulse width control (nanoseconds) for even more fine-tuning

Layer Execution Order

Lightburn runs layers from bottom to top in the Cuts/Layers panel list. This matters:

  • Engrave before cut: put engraving layers lower in the list — they run first. Put cut layers at the top — they run last (after engraving is done on the piece). This prevents the cut piece from shifting mid-engrave.
  • Drag layer rows to reorder them
  • Right-click a layer > "Move to top/bottom" for quick reordering

How Lightburn Engraving Modes Work

Choosing the right engraving mode is as important as choosing the right power/speed. Each mode produces a fundamentally different result.

Line Mode — Vector Engraving

The laser traces each path as a single line. Perfect for stamp plates because:

  • Follows your design paths exactly — no scanning, no fill patterns
  • Clean, precise marks with crisp edges
  • Fast for line-art designs
  • Depth controlled by speed, power, and passes
  • For SS304: use Line mode with 1–3 passes to get the recessed marks stamps need

Fill Mode — Raster Scanning

The laser scans back and forth across the shape's bounding box, firing only when over the shape area — like an inkjet printer.

Overscanning

What is overscanning?

When the laser reverses direction at the ends of scan lines, it slows down (decelerates) and then speeds back up. During this deceleration zone, if power isn't adjusted, the edges of your engraving can be over-burned (the laser dwells longer there).

Overscanning extends the laser's travel beyond the edge of your design so it's at full speed by the time it reaches the engraving. The laser is off during the extended travel.

  • Set in Fill layer settings: Overscan % (typically 2–5%)
  • Higher speed = more overscan needed
  • Too little: burnt edges. Too much: wasted time.

Bi-Directional Scanning

By default, Fill mode only burns on the left-to-right pass and travels back on the right-to-left pass. Enabling bi-directional scanning burns on both passes — roughly 2× faster.

Check for alignment offset Bi-directional scanning can produce slight offset between alternating lines if your machine has backlash or timing issues. Run a test: engrave a fine-detail image and check if lines are perfectly aligned. If not, use the Scanning Offset Adjustment tool (Edit > Machine Settings) to calibrate.

Cross-Hatch Engraving

Run Fill mode twice on the same layer — once at 0° and once at 90°. This creates a cross-hatch pattern that:

  • Engraves deeper than a single pass
  • Looks more uniform — hides linear banding
  • Works well for deep pocket engraving on wood, MDF, and acrylic
  • Enable via: layer settings > Crosshatch checkbox — Lightburn handles the 90° rotation automatically

Combining Line + Fill in One Job

You can have multiple layers with different modes in a single job. Common workflow for stamp logos:

  1. Create your design with all paths on a black layer (Line mode, SS304 settings)
  2. Add a bounding rectangle on a red layer (Line mode, cut settings) — to cut out the stamp blank after engraving
  3. Order layers: black (engrave) runs first, red (cut) runs last
  4. One job = engraved stamp + cut to shape

India Power Supply Note

Protecting Your Fiber Laser in India

Fiber lasers are sensitive to power fluctuations. Indian power supply can have voltage spikes, brownouts, and frequency variation — especially in industrial areas.

  • Fiber lasers are 220–240V: compatible with Indian standard power — no transformer needed
  • Get a UPS (Uninterruptible Power Supply): ₹5,000–8,000 range. Look for pure sine wave output — cheaper UPSes output modified sine wave which can damage laser electronics
  • Recommended UPS: APC Back-UPS Pro, Luminous Sine-wave models, or V-Guard. Minimum 1500VA for a 20W fiber system.
  • Voltage stabilizer: Add a servo-based stabilizer (₹2,000–4,000) if your area has wide voltage swings
  • Earthing: Ensure your workshop has proper 3-pin earthed power. Laser metal frames can become live on floating earth — dangerous and affects mark quality
  • Dedicated circuit: Run your laser on its own MCB (circuit breaker) — don't share with other high-draw equipment

Running a Test Matrix — Step by Step

The power/speed matrix test is the most important step when working with any new material. Do this before every new material, and after any machine maintenance.

1
Prepare material scrap — use a 100×100mm piece of your target material.
2
Open Material Test tool: Tools > Material Test. OR manually create a 5×5 grid of 10×10mm squares, each with a different layer color.
3
Set ranges: Speed axis: 5 steps across your expected range. Power axis: 5 steps from conservative to aggressive. Example for SS304: Speed 100/150/200/250/300 mm/s, Power 40/50/60/70/80%.
4
Run the test — take 5–10 minutes. Ventilate properly. Stay present during the job.
5
Evaluate results: Look for the mark that has the depth and contrast you want. Note the row/column — that's your speed and power target. Write it down and save to Material Library immediately.
6
Repeat for passes: Once you have speed/power, run a second test varying passes (1/2/3/4) to dial in depth for stamp plates.

Focus — Getting the Focal Point Right

Manual Focus (Focus Card / Tool Method)

  • Most fiber/galvo machines ship with a focus card (small metal gauge)
  • Lower Z-axis until card fits snugly between nozzle and material
  • Alternatively: run a ramp test — engrave a line on angled material, sharpest line = focal point
  • Mark the Z height in Lightburn and set as default for that material thickness

Autofocus (if machine supports Z-axis)

  • Some CO2 machines have motorized Z + autofocus sensor
  • Enable in Edit > Device Settings > Z-axis control
  • Set material thickness in job settings — Lightburn adjusts Z automatically
  • Most fiber galvo systems have fixed focal length — manual focus only

Laser Bed Alignment Check

Before precision work, verify the laser head moves square to the bed surface using this simple check:

  • Draw a 100×100mm square in Lightburn
  • Run Frame mode — laser traces the square
  • Use a ruler: measure diagonals of the traced square. If both diagonals are equal, bed is square
  • If not equal: mechanical adjustment needed (gantry squaring, belt tension)
  • For galvo/fiber: use the Field Correction calibration file in Lightburn galvo settings

Optimization Settings Overview

Lightburn's optimization engine controls the order in which the laser executes cuts, how far it travels between cuts, and how it approaches each path. Poor optimization adds minutes of wasted travel time to every job.

Access via: Edit > Device Settings > Additional Settings or the Optimization Settings button in the Laser panel.

SettingWhat it doesRecommended?

Inner before outer — critical for stamp plates If you have text inside a rectangular plate border, Lightburn must engrave the inner text paths before cutting the outer border. Without this setting, the plate border cut causes the piece to shift, making inner details misaligned.

Job Time Estimation

Lightburn estimates job time before you run — check the Laser panel time display after loading a job. This is accurate to within 5-10% for most jobs.

Factors that affect job time

  • Speed: Higher speed = shorter time, but quality may drop
  • Travel distance: Reduce with optimization settings
  • Raster DPI: Higher DPI = more scan lines = more time
  • Number of passes: Each pass multiplies time linearly
  • Acceleration: Machine acceleration limits actual top speed on short segments

Speeding Up Raster Engraving

Increase Scan Speed

  • For CO2: can often run 400–800mm/s for photos
  • For fiber galvo: 1000–3000mm/s is normal for raster fills
  • Test first — going too fast reduces mark depth
  • Compensate by increasing power slightly

Reduce DPI / Line Interval

  • Default 254 DPI → try 200 DPI for quick proofs
  • Line interval: 0.1mm (254 DPI) → 0.15mm (169 DPI) saves ~33% time
  • For stamp plates: 0.05–0.08mm line interval gives sharp detail
  • For photo engraving: 0.08–0.12mm is usually sufficient

Fill Offset — Thickening / Thinning Engraved Lines

In Fill or Offset Fill mode, the Fill Offset parameter expands or contracts the filled area, effectively making engraved lines appear thicker or thinner without changing the original design.

  • Positive offset: expands fill outward — thicker appearance
  • Negative offset: shrinks fill inward — thinner/lighter appearance
  • Use +0.05 to +0.1mm offset for stamp designs that look too light after engraving

Tabs and Bridges

When cutting pieces from sheet material, small parts can fall through or shift before the cut completes. Tabs leave tiny uncut bridges that hold the piece in place.

  • Edit > Add Tabs — set tab width (0.5–1mm) and number per shape
  • After job: snap tabs with a blade or pliers
  • Not commonly needed for engraving-only stamp plates (no through-cuts)
  • Useful if cutting aluminum or acrylic blanks to size with the laser

Lead-In / Lead-Out

Smooth cut entry prevents burn marks or dents at the start/end of a curved cut path.

  • Available in Cut Settings Editor for each layer (Line mode)
  • Lead-in: laser approaches the cut line at an angle before tracing the shape
  • Lead-out: laser exits the cut line at an angle after completing the shape
  • Common for cutting acrylic or wood with CO2 — less relevant for fiber metal marking

Batch Stamp Plate Production Profile

Step-by-Step: Run a Batch of 6 Stamp Plates

India-Specific Problems

Power fluctuation — inconsistent marks Indian mains power (230V) fluctuates significantly, especially in industrial areas during peak hours. Fiber laser PSUs are sensitive to input voltage variation. Solution: use a UPS (online/double-conversion type) rated at minimum 2× your laser's power draw. A 1kVA UPS for a 30W fiber machine is adequate. Servo UPS units (available at ₹8,000–15,000) work well.
Summer heat — thermal drift Fiber laser galvo heads and controllers are temperature-sensitive. In Indian summer (40°C+ ambient), letting the machine warm up unevenly causes the focal point to shift slightly. Solution: power on the machine 30 minutes before precision work; allow galvo head temperature to stabilize before running calibration-critical jobs.
Dust contamination — lens care Workshop dust, especially near grinding or cutting operations, settles on the fiber lens and protective window. Cover the machine when not in use. Clean the protective window (not the main lens) before every session using methanol and lens tissue. A contaminated protective window causes inconsistent power delivery and eventually damages the lens beneath.

  4. Set font size: 12–18pt for a 60×30mm plate
  5. Add a decorative border: draw rectangle, assign to same layer
  6. Use Fill mode for bold text fills: Speed 400mm/s, Power 55%, Freq 30kHz
  7. Use Line mode for outline-only text: Speed 200mm/s, Power 60%, Freq 30kHz
  8. Run on SS304 60×30mm or 60×60mm blank
India business tip: Personalized name plates are high-demand Diwali and wedding gifts. A set of 4 plates (family name + 4 individual names) sells for ₹800–1,500. Production cost: under ₹100. 10-minute job per set. Excellent side-income product.

Project 5 — Heraldry Shield for Warhammer Miniature

Difficulty: Intermediate  |  Time: ~6 minutes  |  Material: SS304 0.3mm thin sheet, 40×40mm

Steps

  1. Download heraldry SVG (free sources: heraldry.drawshield.net, or design in Inkscape)
  2. Import into Lightburn — resize to 40×40mm or smaller as needed for miniature scale
  3. Zoom in — inspect node complexity. Reduce nodes (Edit > Simplify paths) if too complex for scale
  4. Fine detail settings: Speed: 300 mm/s, Power: 50%, Freq: 40 kHz, Passes: 1
  5. Focus extremely carefully — fine detail is very sensitive to defocus
  6. Run on thin SS304 (0.3mm) — lighter material, less thermal mass, use slightly lower power (45%)
  7. Inspect with 10× loupe — lines should be sharply defined at miniature scale

Finished plates can be used as actual miniature shields or as collector display items. Niche but enthusiastic buyer community online.

Project 6 — Anodized Aluminum Business Card

Difficulty: Intermediate  |  Time: ~4 minutes per card  |  Material: Anodized aluminum card blank (black or colored)

How it works

Fiber/MOPA laser removes the anodized layer from aluminum, revealing bright bare aluminum beneath. No chemical processing needed. Result: permanent, bright white or silver text on black/colored card.

Settings (anodized aluminum, not SS304)

ParameterValue
ModeFill (for text and logo areas)
Speed500–800 mm/s
Power25–35%
Frequency50–80 kHz
Line interval0.05mm (508 DPI) for crisp text
Passes1
India business angle: Anodized aluminum cards (blank) available on Amazon.in for ₹20–40 per card. Personalized laser cards sell for ₹150–400 each. Corporate bulk orders (50–200 cards) are very common. Each card takes ~4 minutes → 15 cards/hour → strong revenue-per-hour product.

Steps

  1. Design in Lightburn at 85×54mm (standard card size) with 3mm margin
  2. Include: name, title, phone, email, logo — all as filled text/vector
  3. Place card on bed — use a small jig or tape to fix position consistently for batch runs
  4. Run at anodized aluminum settings above
  5. Wipe with dry cloth — no IPA needed for anodized aluminum

Pro Workflow Tips

Always Frame First

  • Make it a habit: every job, every time — press ~ before Start
  • Takes 5 seconds, saves wasted materials and re-runs
  • Frame confirms position, confirms job is on the material, confirms no obstacles
  • Tape a note to your machine: "FRAME BEFORE START"

Named Material Presets

  • Save verified settings to Material Library (Window > Material Library)
  • Name clearly: "SS304-0.5mm-Outline-30W-MOPA", "Anodized-Al-Fill"
  • Include: speed, power, freq, passes, focus note
  • Export library as backup before any Lightburn update
  • Share library with other machines in your workshop

Focus Card Taped to Machine

  • Your fiber machine's focus card/gauge — tape it to the side of the machine
  • It will get lost otherwise; keeping it attached ensures it's always there
  • Also tape your most-used settings (SS304 outline, fill, deep mark) on a card nearby
  • Label your Z-height markings with a marker directly on the Z column

Job Preview Before Running

  • Ctrl+P opens the Preview window — simulates the entire laser path
  • Scrub through the timeline to verify cut order, travel moves, and path direction
  • Shows estimated time — use this to price jobs accurately
  • Reveals problems (e.g., outer cut before inner details) before any material is used

LightBurn Bridge — Wireless Control

  • LightBurn Bridge is a small device (Raspberry Pi based) that connects to your laser via USB and to your network via WiFi/Ethernet
  • Your PC running Lightburn connects to it wirelessly — laser no longer needs to be physically near your PC
  • Available from lightburnsoftware.com or build your own: ~₹2,500–4,000
  • Useful in Indian workshops where the laser is in a separate room for fume ventilation
  • Works with GRBL, Ruida, and some galvo setups

Update Strategy

Before updating Lightburn:
  1. Export your Material Library: Window > Material Library > Export
  2. Export Device Settings: Edit > Device Settings > Export
  3. Note your current version number (Help > About)
  4. Check Lightburn forum for reports of issues with the new version
  5. Only then: Help > Check for Updates
  6. If new version has problems: Help > Previous Versions to download older installer

Full Business Integration Workflow

From Etsy Order to Shipped Stamp Plate

  1. Receive Etsy order — customer selects design from your listings, provides personalization details (name, text)
  2. Open SVG Studio — load or create the design, apply customer's text/customization, download SVG
  3. Open Lightburn — import SVG, set to customer's required plate size, verify dimensions
  4. Load material preset — apply saved SS304 settings from Material Library (2 clicks)
  5. If batch order — use Array to fill sheet with multiple identical or mixed designs
  6. Frame and run — verify, start, walk away for 5–25 minutes
  7. Inspect and photograph — check quality, photograph for records
  8. Pack and ship — bubble wrap, padded envelope; India Post or courier
  9. Mark order complete on Etsy, upload tracking number
  10. Typical turnaround: same day for orders received before noon; next day for afternoon orders

Advanced: Lightburn API & Automation

  • Lightburn supports command-line job submission — useful for automating batch runs from a production script
  • Combine with Python scripts to auto-generate Lightburn job files from a spreadsheet of customer orders
  • Not officially documented but community scripts exist on the Lightburn forum
  • For high-volume operations (50+ plates/day): worth investing time to automate file generation
Final tip — the single best habit for a stamp plate business Keep a physical logbook next to your laser. For every job: date, design name, material batch, settings used, result quality (1–5 scale), any problems. After 3 months you'll have your own reference book of what works on your specific machine — more valuable than any guide because it's calibrated to your exact setup.
