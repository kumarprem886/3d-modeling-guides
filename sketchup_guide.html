Push/Pull & Follow Me

TipPush/Pull is SketchUp's signature tool. It's intuitive: click any flat face, drag to extrude it into 3D. Double-click repeats the last distance — essential for batch-extruding matching walls.

Architecture / Product Workflow — Step by Step

InfoSketchUp excels at fast concept modeling and architectural design. It's loved for its low learning curve — most users can build their first room in under an hour.
  • 1
    Import Reference: File → Import a reference image (set as "Use as Image" and align to ground plane) or import a DWG floor plan. Lock it in a separate Tag/Layer so it doesn't get edited accidentally.
  • 2
    Draw 2D Floor Plan: Use the Line tool (L) to trace the floor plan outline on the blue/red ground plane. Close all loops — SketchUp automatically fills enclosed areas with a face. This face is what you'll push/pull to create walls.
  • 3
    Push/Pull Walls: Select the floor face, use Push/Pull (P) to extrude it to ceiling height. Type the exact height numerically (e.g., "2800mm" or "9'2""). Use Ctrl+Push/Pull on interior walls to leave the floor face intact.
  • 4
    Add Openings: Draw rectangles directly on wall faces where doors and windows go. Push/Pull these rectangles inward — when you push through to the opposite wall face, SketchUp automatically creates a hole. No boolean needed.
  • 5
    Import Furniture as Components: Use Window → Components to access 3D Warehouse directly. Search for furniture, fixtures, and fittings. Import them as Components so you can swap models later without breaking the scene.
  • 6
    Apply Materials: Open Window → Materials and drag materials onto faces. Use the Texture Adjust option (RMB on textured face) to scale and position textures to real-world size. Match tile/wood grain scale to actual dimensions.
  • 7
    Set Up Scenes: Position camera for each key view. Camera → Add Scene for each viewpoint. Scenes capture camera position, style settings, and tag visibility — use them for client presentation or as render viewpoints in V-Ray/Enscape.
  • 8
    Render or Export: Use Enscape for a real-time walkthrough or V-Ray for high-quality stills. Alternatively, File → Export 3D Model (OBJ/FBX) to take the model into Blender, 3ds Max, or Lumion for advanced rendering.
