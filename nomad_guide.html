Voxel Remesh & Multiresolution

Voxel Remesh Scene → Voxel Remesh
Multiresolution Sculpt → Multiresolution → Add Level
Level navigation < > arrows
Dynamic topology Toggle in brush settings
Decimation Reduce polygon count for export
Tip: Voxel Remesh works like ZBrush DynaMesh — it retopologizes the entire mesh. Use it after big shape changes before adding fine detail.
Tip: Higher voxel resolution gives more detail but produces a larger file size. Start at a low resolution, then increase as you refine.
Tip: Multiresolution levels work just like ZBrush SubDiv — sculpt at different levels non-destructively and navigate between them with the arrows.

Tip: Matcap mode is ideal for sculpting — it gives a rich, shaded preview without the performance cost of full PBR rendering.
Tip: Use HDRI environment lighting for the most realistic renders. Nomad includes several built-in HDRIs — you can also import custom ones.

Workflow — iPad Character Sculpt

1
Import a base mesh (OBJ sphere or humanoid base) or start from Nomad's built-in primitives. Choose a sphere for organic characters or a box for hard-surface models.
2
Enable Symmetry (X axis) from the top-right icon. Start with the Clay and Move tools to block in primary forms — head volume, torso mass, limb proportions.
3
Use Voxel Remesh (Scene → Voxel Remesh) to clean up topology as you work. Increase the resolution progressively — this is equivalent to ZBrush DynaMesh and keeps the mesh clean.
4
Switch to Multiresolution (Sculpt → Multiresolution → Add Level). Add subdivision levels to sculpt finer details — pores, wrinkles, fabric — without destroying your base shape.
5
Apply materials: tap the object, go to the Material tab in the inspector. Set up skin shaders, metallic armor, or fabric using PBR sliders. Add environment HDRI for realistic lighting.
6
Export as GLB (best quality and self-contained) or OBJ for use in Blender, ZBrush, or Substance Painter. Use Scene → Merge All before export to combine all objects.
Pipeline Tip: GLB is the recommended export format — it includes mesh, materials, and textures in a single file. Use OBJ when you need maximum compatibility with older software.
