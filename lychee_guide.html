Workflow — Detailed Miniature
  1. Import miniature STL. Set scale — miniatures are typically 28mm or 75mm height.
  2. Auto-orient using Smart Orient — aim for the highest possible score.
  3. Hollow if large body (saves resin). Add 2mm drain holes at lowest back point.
  4. Generate Smart Supports. Manually add supports to any red island areas.
  5. Select your printer + resin from the library — exposure settings auto-fill.
  6. Slice → Export to USB/SD in correct format for your printer.
  7. Print, wash in IPA (magnetic washer ideal), UV cure 5–10 minutes.
Tip
For miniatures, use 0.025–0.05mm layers and Light supports — the detail payoff is worth the extra time.
