Pathfinder Panel (Window → Pathfinder)

  • Unite — Merge all selected shapes into one combined outline
  • Minus Front — Subtract the front shape from the back shape
  • Intersect — Keep only the overlapping area
  • Exclude — Remove the overlapping area (like XOR)
Bezier & Node Editing
Press A (Direct Selection) to select and manipulate anchor points. Click a path first, then click individual anchors.
Ctrl+click on pathAdd anchor point at that location
Alt+click anchorDelete that anchor point
Shift+CAnchor Point tool — convert smooth↔corner
Ctrl+JJoin two open end points

Shift+F7Open Align panel — align selected anchors

Path Operations

  • Object → Path → Simplify — Automatically reduce anchor count while maintaining shape fidelity
  • Scissors tool (C) — Click on a path at any point to cut it there into two open paths
  • Knife tool — Click and drag across a shape to freehand slice it into separate pieces
  • Convert Anchor Point tool (Shift+C) — Click smooth node to convert to corner, drag corner to make smooth
Effects & Appearance
Shift+Ctrl+F9Open Appearance panel — layer fills, strokes, effects

Appearance Panel (Core Workflow)

  • Stack multiple fills and strokes on a single object — each with its own effect
  • Add effects per fill/stroke (not just per object) for complex styles without extra layers
  • All effects are live (non-destructive) — click any entry to edit or remove

Effects Menu

  • Effect → 3D & Materials → Extrude & Bevel — Basic 3D extrusion directly in Illustrator
  • Offset Path — Object → Path → Offset Path (grow/shrink shape outline)
  • Pattern — Object → Pattern → Make (create seamless tileable pattern)
  • Envelope Distort — Object → Envelope Distort → Make with Mesh / Make with Top Object (warp shapes)
  • Gradient Mesh — Object → Create Gradient Mesh (complex smooth color transitions)

Symbols

  • Window → Symbols panel — create reusable instances of artwork
  • Select art → drag into Symbols panel to define
  • Place instances with Symbol Sprayer tool — all update when master is edited

Workflow — Texture Art for 3D
This workflow produces high-resolution texture assets for Substance Painter decal layers or Blender shader nodes.
1
Set document at texture resolution — New Document → Width 4096 px, Height 4096 px, 72 DPI (pixels are what matter for GPU textures, not DPI).
2
Design the decal or label — Use Pen tool for custom paths, Shape Builder (Shift+M) to merge geometry, Pathfinder for boolean operations.
3
Use Appearance panel — Stack fills and strokes to create complex multi-layer effects on a single vector object without creating extra layers.
4
Export as PNG — File → Export → Export As → PNG. Set resolution to full document size. Uncheck "Use Artboards" if you want the whole canvas. No compression for maximum fidelity.
5
Import into Substance Painter — Use the PNG as a decal layer in the Texture Set. Scale and position it on the UV map using the 3D viewport.
6
SVG → Blender — File → Save As → SVG format. In Blender: File → Import → Scalable Vector Graphics (.svg). Shape arrives as curve objects. Object → Convert → Mesh to make it editable geometry.
