Layer System

How XCS Layers Work

XCS uses a color-based layer system. Each unique color in your design becomes a separate processing layer, each with its own Speed, Power, Passes, and Mode settings.

  • Black objects → Layer 1
  • Red objects → Layer 2
  • Blue objects → Layer 3
  • ...and so on

This is extremely powerful for multi-step jobs — for example: engrave first (black layer), then cut (red layer), automatically in the correct order.

Layer Processing Panel (Bottom)

Each layer row shows:

  • Color swatch — the color this layer controls
  • Mode — Score / Fill / Cut
  • Speed — mm/s
  • Power — percentage
  • Passes — number of repetitions
  • DPI (Fill mode) — resolution of raster fill
  • Eye icon — toggle visibility
  • Lock icon — lock layer from editing

How to Import Files

Method 1: Drag & Drop (Fastest)

Open your file manager (Windows Explorer / Finder), find your SVG, PNG, or DXF file, and drag it directly onto the XCS canvas. XCS auto-detects the file type and imports it.

Method 2: File Menu Import

File → Import → browse to your file → Open. Works for all supported formats.

Method 3: Copy-Paste from Inkscape

Select objects in Inkscape → Ctrl+C → click XCS canvas → Ctrl+V. Pastes as SVG paths. Useful for quick transfers.

SVG Import — The Key Format for Stamp Plates

Why SVG is Ideal

SVG is a native vector format — XCS reads it directly without rasterizing. This means:

  • Crisp, smooth paths at any size — no pixelation
  • Each stroke/path becomes a separate laser pass
  • Colors map directly to XCS layers
  • Resize freely without quality loss
SVG from Stamp Plate SVG Studio app: Download your SVG → drag into XCS → resize to plate dimensions (e.g., 40×60mm or 60×60mm) → set layer to Score mode → engrave. No intermediate conversion needed — this is the cleanest workflow.

Image Processing Modes (PNG/JPG)

When you import a raster image, XCS asks you to choose a processing mode. This determines how the image is converted into laser movements:

ModeHow It WorksBest For
Greyscale Maps pixel brightness to laser power — darker pixels = more power Portrait photos, detailed grayscale artwork on wood/acrylic
Black & White Threshold converts to pure black/white — pixels are either engraved or not Logos, high-contrast line art, text from images
Seal Inverts image for stamp plate use — dark areas become raised rubber Stamp plate images (not needed if using SVG)
Line Edge detection — only engraves the outlines/edges of shapes Sketchy engraved look, portraits with line-art effect

Tracing Images to Vector

1
Import your PNG or JPG image onto the canvas.
2
Right-click the image → Trace Image (or Image menu → Trace).
3
Adjust the Threshold slider — higher = more detail, lower = simpler paths.
4
Click Apply — XCS generates vector paths from the image.
5
Delete the original raster image — you now have clean vector paths for laser engraving.
Trace quality depends on image quality. Use high-contrast PNG images for best results. Photos with gradients trace poorly — use Greyscale mode for those instead.

DXF Import Tips

  • DXF from Fusion 360: Export as DXF, set units to millimeters in Fusion before export
  • DXF from AutoCAD: Save as DXF 2013 or older — newer DXF versions may not import
  • After import, always check the size: if design is 25.4x too large, the DXF was in inches — scale down by ÷25.4
  • DXF layers become XCS color layers (layer names/colors should be set in CAD software)

Resizing and Positioning Imported Files

Visual Resize

  • Click object → drag corner handles
  • Hold Shift while dragging to lock aspect ratio
  • For stamp plates: enable lock ratio, then set exact width or height

Exact Dimensions

  • Select object → type W and H values in Properties panel (right)
  • Lock ratio padlock ensures proportional scaling
  • For 40×60mm stamp plate: set W=40, H=60 (verify ratio locked)

Best Practices for Stamp Plate SVG Files

  • Use stroke paths only (no fills) for text and outlines — maps cleanly to Score mode
  • Set document size in Inkscape to match your plate (40×60mm or 60×60mm) before saving
  • Convert all text to paths (Path → Object to Path) before export
  • Use a single color (black) for all paths — XCS will assign to one layer, easy to set Score mode on all at once
  • Group the entire design: Ctrl+A then Ctrl+G before export
  • Save as Plain SVG (not Inkscape SVG) for best compatibility with XCS

The Three Processing Modes

Score (Vector Engrave)

The laser follows the exact path of your vector lines — like a pen drawing. Used for outlines, text strokes, detailed linework.

Stamp plates: Score mode is what you want. Each stroke becomes a precise groove in the metal.

Follows paths Fast Stamp plates

Fill (Raster Engrave)

The laser sweeps back and forth (like a printer) filling solid areas. Used for logos, filled text, photos, and solid shapes.

DPI setting controls resolution — higher DPI = more detail but longer time.

Raster scan Slower Logos/fills

Cut

Laser moves along the path with high enough power to cut completely through the material. Usually slower, higher power, multiple passes.

Note: xTool F1 Ultra fiber laser cannot cut — it marks/engraves. The 2W diode can cut thin materials.

Cuts through High power

Layer Order & Processing Sequence

XCS processes layers from bottom to top in the layer list. This is important for multi-operation jobs:

  • Engrave first, cut last — if you cut first, the material may shift, misaligning the engrave
  • Drag layers in the bottom panel to reorder them
  • Standard workflow: engrave layers at bottom → score/detail layers in middle → cut layer at top

The Two Lasers — What Each Does

20W MOPA Fiber Laser

Metal marking Permanent Stamp plates

Best for:

  • Stainless steel (SS304, SS316) — stamp plates, cutlery, tools
  • Aluminum — anodized (colors) and bare
  • Brass, copper, gold, silver, titanium
  • Some plastics and coated surfaces

Cannot do: Wood, acrylic, leather, fabric, paper (use diode)

MOPA advantage: Adjustable pulse width — enables color annealing on stainless steel (black, blue, gold, etc.) by changing pulse settings

2W Blue Diode Laser

Organic materials Wood/leather Limited cutting

Best for:

  • Wood engraving (teak, walnut, bamboo, MDF)
  • Leather engraving (tags, wallets, journals)
  • Acrylic engraving (frosted look)
  • Fabric / felt marking
  • Paper cutting (thin sheets)
  • Dark coated materials

Cannot do: Bare metal, transparent acrylic cutting

Note: 2W is lower power than dedicated diode machines — for deep cuts, use a dedicated S1 or D1 Pro

The F1 Ultra Stamp Plate Workflow — Step by Step

1
Design: Create your stamp design in SVG Studio app (or Inkscape, Procreate, etc.) → export/download as SVG file.
2
Import: Open XCS → drag your SVG file onto the canvas. The design appears as a vector layer.
3
Resize: Select the design → lock aspect ratio in Properties → type exact plate dimensions (W: 40, H: 60 for a 40×60mm plate, or W: 60, H: 60 for square).
4
Mirror: Click Flip Horizontal in Properties panel. Stamps must be mirrored — what's engraved is the reverse of the final impression.
5
Layer settings: In the bottom panel, set Mode to Score, select Stainless Steel from material presets. Starting settings: Speed 200 mm/s, Power 55–60%, Passes 1.
6
Preview: Click Preview → verify the laser path looks correct → check time estimate (should be 30–90 seconds for a typical stamp plate).
7
Position: Use Camera View to visually place the design over your SS304 plate on the bed. Or use the corner/edge positioning method with the F1 Ultra's built-in framing function.
8
Engrave: Place SS304 plate on the bed, close the lid, click Start in XCS. The F1 Ultra auto-focuses, then runs the job. Done in 30–90 seconds.
9
Clean: Open lid, remove plate. Clean with IPA (isopropyl alcohol) and a cotton swab to remove metal dust and reveal the clean engraving.
10
Test: Press the stamp plate onto a stamp pad and test impression on paper. Adjust depth on next plate if needed (reduce speed or increase passes for deeper marks).

Advanced Techniques

▦ Layering Multiple Passes for Deep Engraving

Single-pass engraving sits at roughly 0.1–0.3 mm depth on wood. For 3D-relief plaques, jewellery boxes, or architectural models you need controlled multi-pass layering.

1
Design in layers. In XCS, duplicate your artwork across separate layers — each layer gets its own power/speed setting. Name them Pass-1, Pass-2, etc.
2
Increment power, keep speed. Start Pass-1 at 60 % power, Pass-2 at 70 %, Pass-3 at 80 %. Keeping speed constant gives uniform char removal per pass.
3
Air assist on. Between passes, debris accumulates. Air assist clears it and prevents flare-ups that cause uneven depth.
4
Focus check. As material ablates, focal distance changes slightly. For >3 passes on acrylic, re-focus between pass 2 and 3.
💡 Tip: Do NOT move the material between passes. Lock your origin in XCS or use registration holes drilled at corners of your stock.

🌕 Variable Power Gradients for 3D Effect

Simulate 3D depth using a grayscale bitmap where pixel brightness maps directly to laser power — white = full power (deep), black = no power (surface).

Workflow in XCS

  1. Create/import a grayscale height-map image (PNG, 300 DPI).
  2. Set layer mode to Grayscale (not dithered).
  3. Set max power 85 %, speed 800 mm/min, lines/inch 254.
  4. Run a test tile — adjust gamma curve if midtones are too flat.
  5. Post-process: sand with 400-grit, apply Danish oil for contrast.

Image Prep Tips

  • Use Photoshop/GIMP: Image → Mode → Grayscale
  • Add 10–15 % contrast boost — lasers compress midtones
  • Blur edges 1–2 px to avoid banding
  • Invert if your software expects inverted brightness
  • Canvas size = actual burn area in mm at 10 px/mm
📷 Best materials for gradient engraving: Anodized aluminium (dramatic), Baltic birch, dark leather. Avoid MDF (tears up) and acrylic (melts unevenly).

🏭 Multi-Material in a Single Job

You can engrave two different areas with different settings in one session by masking unused zones with painter's tape or transfer tape.

1
Lay material flat. Tape adjacent materials flush to the bed — e.g., leather left, wood right.
2
In XCS, assign layers by zone. Elements over the leather zone → Leather layer (speed 200, power 50 %). Elements over wood → Wood layer.
3
Run each layer separately. Pause between, adjust air assist if needed.
4
Masking tape as protective layer. Apply tape on the surface you don't want smoke-stained, then remove after the job — especially useful on mirror acrylic.
⚠ Never mix materials that off-gas dangerously near each other (PVC near food-safe wood). Keep incompatible materials in separate jobs.

📷 Camera Alignment for Precise Repeat Positioning

xTool S1, D1 Pro and F1 Ultra have built-in cameras. Calibrate once; thereafter the camera overlays a live preview on your stock so you can position artwork precisely.

Calibration Procedure

  1. Open XCS → Camera → Calibrate Camera.
  2. Place the printed calibration card (from xTool package) on the bed.
  3. Click Start Calibration. The laser marks 9 target points.
  4. XCS asks you to click on each mark in the camera image — do this carefully; accuracy here determines overlay precision.
  5. Save calibration. Typical accuracy: ±0.5 mm at bed centre, ±1 mm at corners.

Repeat Positioning Workflow

  1. Place item on bed. In XCS, capture a camera frame.
  2. Drag your artwork over the camera image to align visually.
  3. For repeat batches: use Array Grid (see next section) and place a row of identical items.
  4. Camera overlay shows exact placement — adjust until artwork sits over the item.
⚠ Re-calibrate after moving the machine or adjusting bed height. Camera accuracy degrades if the machine is physically disturbed.

☷ Batch Processing & Array Grid Function

Array Grid lets you tile your design across a grid of items — perfect for engraving 10 keychains, 20 coasters, or 50 labels in one job.

Using Array Grid in XCS

  1. Select your artwork.
  2. Object → Array Grid (or Ctrl+Shift+A).
  3. Set rows, columns, X-spacing, Y-spacing.
  4. Spacing = item width + gap (usually 2–5 mm).
  5. Preview the grid. Confirm it fits bed.
  6. Run job — all copies engraved in one pass.

Batch Tips

  • Use jig (MDF template with cut-out pockets) so items stay aligned across batches.
  • Mark jig origin with a small laser dot — re-home to same point each load.
  • For odd-shaped items (keychains), cut the jig pockets first, then switch to engrave job without moving anything.
  • Batch mode in XCS lets you load a CSV of names/text to auto-personalise each copy.

▭ Framing Before Cut & Tab/Bridge Function

Framing (Outline Preview)

Before any cut job, always run Frame (the bounding box preview). The laser head traces the outer rectangle of your job at low power — no burning. This confirms:

  • Job fits within the material boundaries.
  • Origin point is where you expect.
  • Rotary/fixture won't collide with machine walls.

Shortcut: F key or the Frame button in XCS toolbar. For shaped outlines (not just bounding box), use Trace Frame mode.

Tabs / Bridges for Cut Pieces

When cutting small pieces, they fall through or shift before the cut completes — ruining the edge. Tabs are small uncut bridges that hold the piece in place.

  1. Select cut path → right-click → Add Tabs.
  2. Set tab width: 1–2 mm for thin material, 3 mm for 6 mm+ wood.
  3. Set tab count: 2 tabs for pieces <50 mm, 4 tabs for larger pieces.
  4. After job completes, snap pieces out by hand. Sand tab stubs with 220-grit.
💡 For acrylic, tabs leave small marks — finish with a flat scraper or file. Alternatively, apply masking tape on both sides before cutting; it acts as a holder and protects the surface from smoke.

🇮🇳 India-Specific Issues

⚡ Power Fluctuation Causing Mid-Job Failure

Indian power grids — especially in Tier 2/3 cities — experience frequent voltage fluctuations (180 V to 260 V). A sudden voltage drop mid-job causes the machine to restart, ruining expensive material.

Fix:
  • Use a 1 kVA Online UPS (APC or Luminous brand, ₹8,000–15,000). Online UPS (not offline/standby) provides zero-transfer-time switchover.
  • Add a Servo Voltage Stabilizer (1 kVA) — ₹3,000–5,000 — if UPS budget is too high.
  • Check machine input voltage range: xTool D1 Pro accepts 100–240 V. F1 Ultra: 100–240 V. Still, surges >260 V can damage SMPS.
  • Add a surge protector strip as a minimum — ₹500–800 at local electrical store.
☀ Summer Heat and Machine Drift (April–June)

Ambient temperatures of 38–45 °C in Indian summers cause thermal expansion of the machine frame, leading to focus drift and accuracy loss.

  • Run machine in an air-conditioned room, or use a room cooler to keep ambient below 32 °C.
  • Warm machine up for 10 min before production work — allows frame to reach thermal equilibrium.
  • Re-calibrate camera every morning during summer.
  • For the F1 Ultra fiber laser: the internal chiller handles laser tube cooling, but the motion system still drifts — keep room cool.
🌧 Monsoon Humidity Affecting Wood (June–September)

Relative humidity of 80–95 % causes wood to absorb moisture, swelling by 1–3 % and increasing char. Engravings appear darker and cuts need more passes.

  • Store wood stock in a sealed plastic bin with silica gel packets (₹50 for 10 packs on Amazon).
  • Dry wood pieces in an oven at 60 °C for 30 min before lasering if very humid.
  • Re-run your material test tiles during monsoon — presets from winter will burn darker.
  • Acrylic and metal are unaffected — shift to these materials for mission-critical jobs during peak monsoon.

📋 How to Build Your Own Preset Library

MATERIAL TEST TILE PROCESS
==========================
1. Cut a 100×100 mm piece of the new material.
2. Create a 5×5 grid of 18×18 mm squares in XCS.
3. Label columns: Speed 50, 100, 200, 400, 800 (mm/min)
4. Label rows: Power 20%, 40%, 60%, 80%, 100%
5. Each square engraves at that speed/power combo.
6. Run the test. Examine each square.
7. Circle the best result. Note settings as your preset.
8. Photograph the tile and file it — future reference.

For cutting tests:
  - Draw 5 lines at different speeds, same power
  - Check which cuts fully through in 1 pass
💡 Save your test tiles! Glue a label with material source, date, and machine settings. Build a physical binder of test tiles — invaluable when a material batch changes.

🔄 Project Sharing — XCS File Format

XCS saves projects in .xcs format (ZIP-based archive containing SVG layers + settings JSON). You can share these files with other XCS users.

Sharing via Google Drive

  1. In XCS: File → Save As → choose location. Extension: .xcs
  2. Upload to Google Drive folder → right-click → Get Shareable Link → "Anyone with link can view".
  3. Share link in Facebook groups or WhatsApp — recipients download and open in XCS.
