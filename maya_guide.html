VFX / Film Pipeline — Step by Step

InfoThis workflow follows the standard character animation pipeline used in VFX and animated film production.
  • 1
    Model & UV Unwrap: Build clean polygon geometry. Keep quads only — avoid triangles and n-gons. Open the UV Editor (UV → UV Editor) to unwrap the mesh so textures apply correctly. Freeze transforms and delete history.
  • 2
    Rig with Joints: Create the skeleton using the Joint Tool (Skeleton → Create Joints). Set up IK handles for the limbs. Create control curves (circles) to act as animatable control objects — constrain joints to controls, not animate joints directly.
  • 3
    Skin Binding: Select the mesh and all joints, then Skin → Bind Skin → Smooth Bind. Use default max influences of 4. The mesh is now deformed by the skeleton.
  • 4
    Paint Weights: Open Skin → Paint Skin Weights. Paint each joint's influence area carefully. Start with Flood at 0 to clear, then build up. Check deformations by posing the rig as you paint.
  • 5
    Dynamics (optional): Add nCloth for fabric, nHair for hair/fur dynamics. Cache simulations to disk for playback performance. Simulations must be cached before rendering.
  • 6
    Arnold Materials & Lighting: Assign aiStandardSurface shaders. Set up a skydome_light with HDRI for environment. Add aiAreaLights for key/fill/rim lighting. Enable IPR for interactive render preview.
  • 7
    Render AOV Passes: Set up AOV render passes in Render Settings. Render to EXR format for maximum dynamic range. Composite in Nuke or After Effects: beauty + diffuse + specular + shadow passes combined give full control in post.
