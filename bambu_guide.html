Workflow — Multi-Color Print

  1. Import STL. Select your printer (X1C, P1S, A1 Mini, etc.).
  2. Right-click model → Color Painting. Paint each region a different filament color.
  3. Check wipe tower placement — move it away from the model if needed.
  4. Adjust flush volumes: higher = less color bleed, more waste filament.
  5. Slice (G). Check Preview tab — verify color changes appear correct on each layer.
  6. Send to printer via WiFi or save to SD/USB.
  7. Load AMS filament in correct order matching your color assignments.
Tip: Always verify color assignments in the Preview tab before sending to printer — misassigned filaments waste an entire AMS spool.
