Orientation & Hollowing

Danger: Orientation is CRITICAL for resin — wrong angle causes large suction forces and print failure.
Recommended tilt angle15–45°
Hollow wall thickness2–3mm recommended
Drain holes when hollow2 holes at lowest point
Tip: Rotate so fine details face up (away from build plate) for highest quality surface finish.
Warning: Always add drain holes when hollowing — trapped resin inside can cause pressure buildup and cracking.
Shell/hollow mode reduces resin cost significantly — expensive resin adds up fast on large prints.

Support Settings

Auto support accessAnalysis → Auto Support
Support density0.5–1.5 per cm²
Support typesLight / Medium / Heavy
Support top radius0.2–0.3mm (smaller = easier removal)
Support bottom radius0.5–1mm (stability)
Raft thickness5–8mm (always use for resin)
Warning: Island detection: Analysis → Islands. Red areas = unsupported floating geometry — must add supports there.
Tip: After auto-support, always run island detection and manually add supports to any remaining red areas.
Danger: Never skip the raft for resin prints — without it the model cannot adhere to the build plate.

Normal exposure time2–5 seconds per layer
Lift speed — standard1–3mm/s

Lift height5–7mm
Danger: Exposure too high = overexposure (lost fine detail). Too low = failed/delaminating layers.
Warning: Always start with your resin brand's recommended exposure settings — they're tested for your UV power.
Tip: Enable anti-aliasing for smoother curves and edges on detailed models.

Workflow — Resin Print

  1. Import model (STL/OBJ). Check scale — resin printers have smaller build volumes than FDM.
  2. Orient model: tilt 15–45°. Minimize large flat surfaces parallel to build plate.
  3. Hollow if large solid model (saves resin). Add 2 drain holes at lowest point.
  4. Run Auto Support. Add manual supports to any red-highlighted islands.
  5. Set layer height (0.05mm standard). Check exposure settings for your resin brand.
  6. Slice → save CTB/photon/cbddlp file to USB.
  7. Print. Post-process: IPA wash 60–90 seconds, then UV cure 2–5 minutes.
Danger: Always wear nitrile gloves when handling uncured resin — it is a skin sensitizer and can cause allergic reactions with repeated exposure.
Tip: Use a magnetic wash station with IPA for faster, cleaner post-processing of resin prints.
