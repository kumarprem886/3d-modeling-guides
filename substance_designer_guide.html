Key Nodes

Generators

Noise base grayscale noise patterns
Tile Generator repeating tile patterns
Gradient linear/radial gradients
Waveform sine/square wave patterns
Splatter scattered shape distribution
Polygon geometric polygon shapes

Filters

Blur soften/blur texture
Sharpen increase edge contrast
Levels remap value ranges
Histogram Scan precise value range control
Warp distort with another texture
Slope Blur blur along slope direction

Blending & Transform

Blend combine two inputs (multiply, add, overlay, etc.)
Transform 2D move/rotate/tile a texture
Tile Sampler place instances across UV space
Pattern create repeated geometric patterns

Normal & Output

Normal (from height) convert height to normal map
Normal Combine merge two normal maps
RGBA Merge/Split combine or split color channels
Output nodes define what textures export

Procedural Workflow

1
Start with Generators (noise, cloud, tile) for base shapes
2
Blend/mask generators to create varied base pattern
3
Use Slope Blur and Warp for natural-looking irregularity
4
Histogram Scan to control exact range of values
5
Add detail with Grunge maps from Substance Library (1000s available)
6
Create separate height, roughness, and base color graphs (or one unified graph)
7
Connect to Output nodes and publish as .sbsar
Substance Library has thousands of grunge/noise maps ready to use as inputs. Build modular graphs — one for base, one for detail.

Workflow / Create Tileable Material

1
New graph → blank. Add Output nodes for BaseColor, Roughness, Metallic, Normal, Height
2
Generate base: add Tile Generator → set pattern (brick, stone, etc.)
3
Add noise variation: Clouds node → Warp the tile pattern for irregularity
4
Histogram Scan to create clean black/white mask for mortar/grout areas
5
Blend colors: Uniform Color for stone + Blend with Grunge for variation
6
Build Roughness: stones = low roughness (smooth), mortar = high roughness
7
Height map drives Normal map: use Normal node from height → connect to Normal output
8
Export Substance Archive (.sbsar) or PNG maps at any resolution
