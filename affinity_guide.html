Navigation & Personas
3 Personas (Modes): Affinity Designer switches between Vector, Pixel, and Export personas — each unlocks a different set of tools.

Switching Personas

  • Vector Persona — Pen tip icon top-left. Draw and edit vector paths, shapes, boolean operations.
  • Pixel Persona — Brush icon top-left. Paint pixels, use raster brushes and filters.
  • Export Persona — Arrow-box icon top-left. Set up export slices and presets for batch export.

Navigation

  • Scroll wheel = Zoom in/out on canvas
  • Middle Mouse Button drag (MMB) = Pan canvas
  • Space + drag = Pan canvas (alternate — works with any tool active)
  • Ctrl+0 = Fit entire document to window
  • Ctrl+1 = 100% zoom (actual pixels)
  • Ctrl+2 = 200% zoom, Ctrl+3 = 300% zoom

Essential Tool Keys

  • V = Move tool — select and reposition objects
  • A = Node tool — edit bezier anchor points and handles
  • P = Pen tool — draw new paths and shapes

Boolean Operations

  • Add — Layer menu → Geometry → Add. Merges selected shapes into one united shape.
  • Subtract — Cuts the top shape out of the bottom shape.
  • Intersect — Keeps only the overlapping area.
  • Divide — Cuts shapes into separate pieces at intersections.
  • Combine — Merges paths as compound path (keeps overlap holes).
  • XOR — Keeps non-overlapping areas, removes intersections.

Other Operations

  • Knife tool — Click and drag across a shape to slice it into separate pieces
  • Add Artboard — Document menu → Insert Artboard (or A key in toolbar mode)

Node Types (toolbar buttons when Node tool active)

  • Sharp (Corner) node — Handles move independently. Click the sharp node icon in top toolbar.
  • Smooth node — Handles stay collinear (opposite). Click smooth icon in top toolbar.
  • To convert: select a node, then click the node type button in toolbar.

Layer FX (Non-Destructive Effects)

  • FX Panel — Drop Shadow, Inner Shadow, Outer Glow, Inner Glow, Outline, Color Overlay, Gradient Overlay, Bevel/Emboss. All are non-destructive and editable.
  • Blend Modes — Full Photoshop-compatible list in the Layers panel: Normal, Multiply, Screen, Overlay, Soft Light, Hard Light, Color Dodge, etc.

Symbols

  • Create reusable components: select shapes → Layer → Create Symbol
  • Manage in Assets panel → Symbols tab
  • Edit one Symbol instance → all instances update automatically

Styles

  • Save a consistent look (fill + stroke + FX) as a named Style
  • Open Styles panel → click Add Style with object selected
  • Apply to any object by clicking the Style thumbnail

Adjustment Layers

  • Layer menu → New Adjustment Layer → Curves / Levels / HSL / Brightness-Contrast / etc.
  • All adjustments are fully non-destructive and reorderable
  • Clip an adjustment to a single layer using Alt+click between them

Workflow — Logo for 3D Decal/Texture
This workflow produces export-ready assets for use as decals in Substance Painter or as imported curves/meshes in Blender.
1
Set document units to mm — match the units of your 3D model's UV space for accurate scale. File → Document Setup → Units.
2
Draw logo shapes — Use the Pen tool for custom paths and geometric primitives (M, O, Shift+M) for common shapes. Work in Vector Persona.
3
Apply boolean operations — Layer → Geometry → Add / Subtract / Intersect to combine or subtract shapes cleanly. No stray overlap geometry.
4
Assign flat colors only — Avoid gradients for clean decal use. Flat fills export predictably and look correct when projected onto 3D surfaces.
5
Switch to Export Persona (arrow-box icon) → Add export preset → PNG at 4096×4096 px for maximum resolution, or SVG for vector import into Blender.
6
Import into 3D software — PNG into Substance Painter as a decal sticker layer, or import SVG into Blender via File → Import → SVG (imports as curve objects, convert to mesh as needed).
