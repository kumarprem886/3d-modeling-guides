Tip: 3D Coat has 4 main rooms: Sculpt, Retopo, UV, and Paint. Select the active room from the top bar. Each room has its own tool set and panels.

Sculpt Room

Add primitive or import mesh Shift + A
Smooth brush Shift + LMB

Layers panel Add layer for non-destructive sculpting
Voxel sculpting Unlimited boolean-like sculpting
Surface sculpting Like ZBrush, works on mesh surface

Tip: Import reference meshes via top menu → Import → Import Mesh. Use voxel sculpting for destructive boolean operations and surface sculpting for fine detail work.
Tip: Voxel mode treats the mesh like a volume — you can freely add and subtract without worrying about topology. Switch to Surface mode for ZBrush-style detailed sculpting.

Standout Feature: The Retopo Room is 3D Coat's crown jewel. It lets you draw clean quad topology directly on top of your high-poly sculpt, then bake all the detail down to a normal map — the complete game-asset workflow in one tool.

Workflow — Game Asset Pipeline

1
Sculpt the high-poly mesh in the Sculpt Room using voxel or surface sculpting mode. Alternatively, import a high-poly from ZBrush or Blender via top menu → Import → Import Mesh.
2
Switch to the Retopo Room (F6). Use Auto Retopo (Retopo → Autopo) for a fast result, or manually draw clean quads using the Draw Faces and Quads Draw tools. Target your game polygon count.
3
UV unwrap in the UV Room (F7). Mark seams on natural breaks (joints, edges, hidden areas), run Unwrap, then Pack UVs to fill the UV space efficiently. Check for stretching with the texture checker.
4
Bake the high-poly detail to the low-poly mesh using the Bake menu. This generates a normal map that transfers all the sculpted detail onto the clean retopologized mesh.
5
Paint textures in the Paint Room (F8): apply diffuse color, roughness, metalness, and emissive maps simultaneously. Use Smart Materials for realistic surface presets and stencils for surface detail.
6
Export the full texture set at 4K resolution (PNG or TGA) via File → Export Textures, plus the low-poly mesh as OBJ. Import into Substance Painter for further texturing or directly into your game engine.
Pipeline Tip: 3D Coat covers the entire game-asset pipeline in one application — sculpt, retopo, UV, bake, paint. Mastering the room-switching workflow (F5–F8) is the key to a fast, professional pipeline.
Export Tip: Export a 4K PBR texture set: Diffuse (albedo), Normal, Roughness, Metalness, and optionally AO and Emissive. These map directly to standard PBR slots in Unity, Unreal, and Blender.
