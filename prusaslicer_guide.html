Workflow — First FDM Print

  1. Open PrusaSlicer. Select your printer profile (Prusa i3, MK4, Mini, or generic FDM).
  2. Add model: drag STL onto plate or press Ctrl+I to import.
  3. Orient model: rotate so flat face is on bed. Minimize overhangs.
  4. Set layer height (0.2mm standard), Infill (15–20%), Perimeters (3).
  5. Enable support if model has overhangs over 45°.
  6. Slice (right side → Slice Now). Check preview layer by layer.
  7. Export G-code to SD card/USB. Print!
TipAlways preview with the layer slider before exporting — catch problems before wasting filament.
