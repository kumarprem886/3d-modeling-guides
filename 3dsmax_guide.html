Rendering with V-Ray

InfoV-Ray by Chaos Group is the dominant renderer for architectural visualization and product rendering. It's a biased renderer that balances quality and speed for production.

Distributed network renderV-Ray DR (Distributed Rendering)
TipUse the V-Ray Frame Buffer LUT feature to apply a cinematic look (ACES, filmic) directly in the VFB without needing Photoshop post-processing.
WarningAlways use V-Ray Physical Camera instead of the standard Max camera when rendering with V-Ray — the exposure, DOF, and motion blur work in real-world units and prevent blown-out renders.

Arch-Viz Pipeline — Step by Step

InfoThis workflow covers a typical interior architectural visualization project from CAD import to final deliverable.
  • 1
    Import Floor Plan: File → Import the AutoCAD DWG. Lock the DWG layer and use it as a tracing reference. Set units to match the drawing (millimeters or meters).
  • 2
    Model Walls: Use the Wall object from AEC Extended to trace the floor plan outline, or draw splines over the plan and extrude to wall height. Keep each room as a separate element for easy material assignment.
  • 3
    Add Doors & Windows: Use AEC Extended door/window objects for parametric openings, or model custom openings by drawing rectangles on the wall mesh and using ProBoolean to cut through.
  • 4
    Import Furniture: Download high-quality assets from Evermotion, CGAxis, or 3D Warehouse. Import as XRef or merge. Use ProxyMesh for heavy objects to keep viewport fast.
  • 5
    Apply V-Ray Materials: Set up materials in the Slate Material Editor. Use V-Ray materials with proper IOR values (glass = 1.52, chrome = 0.04 roughness). Add displacement maps for carpet and stone.
  • 6
    Place Camera: Create a V-Ray Physical Camera at eye level (900-1600mm). Set F-stop (f/8 for most interiors), ISO (800-1600), and shutter speed. Enable DOF for foreground blur if needed.
  • 7
    Render & Post-Process: Render at high resolution (3000-5000px wide). Use the V-Ray Frame Buffer to apply color correction and LUT. Export to Photoshop for final touch-ups, or use VFB compositing directly for faster iteration.
