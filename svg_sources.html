▶ From SVG to Finished Stamp Plate

Step-by-step: download from sources above → arrange in Inkscape → send to your laser engraver.

WHAT YOU NEED
Inkscape (free — inkscape.org) SVG files from sources above SS304 mirror finish steel plate Fiber laser + EzCad / Lightburn / xTool XCS
1
Set up your plate canvas in Inkscape
File → Document Properties (Shift+Ctrl+D) → set Units to mm → set canvas to your plate size.
Common sizes: 40×60mm   60×90mm   100×150mm   60×60mm square
2
Import and clean each SVG
File → Import (Ctrl+I) → select SVG.
Then: Ctrl+A → Path → Object to Path → Ctrl+Shift+G (ungroup, repeat until flat).
Open Fill & Stroke (Shift+Ctrl+F): Fill → No paint, Stroke → Black #000000, width 0.1mm.
Delete any white background rectangles. Then Path → Combine (Ctrl+K) to merge all paths.
3
Arrange designs on the plate
Import each cleaned SVG with Ctrl+I. Resize holding Ctrl+Shift + drag corner (proportional).
Leave 2mm gap between designs, 3mm from edges.
Use Align & Distribute (Shift+Ctrl+A) to space evenly.
Typical layout for 60×90mm: 3 columns × 4 rows = 12 designs.
4
Export the final plate file
File → Save As → Plain SVG (for Lightburn / xTool XCS).
File → Save a Copy → DXF (R14 format) for EzCad.
Name clearly: plate_warhammer_60x90mm.svg
5
Send to laser software
Lightburn: File → Import SVG → Layer mode: Line → Speed 200mm/s, Power 60%, Freq 30kHz, 1 pass → Frame → Start
EzCad: Import DXF (R14) → Speed 200, Power 60%, Freq 30kHz, Q-Pulse 100ns, Loop 1 → F2 to engrave
xTool XCS: Import SVG → mode: Score → Fiber module → Speed 200, Power 60%, Freq 30kHz, 1 pass → Process
6
Finish the plate
Engrave shiny side up (mirror finish faces the laser).
After engraving, wipe with isopropyl alcohol → marks turn crisp and sharp.
Test settings on scrap SS304 before running the full plate.
