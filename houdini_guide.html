VEX & Python Basics

InfoVEX (Vector Expression Language) is Houdini's geometry/shader language. It runs per-point/edge/primitive in an Attribute Wrangle node. It's fast and parallelized.
Key VEX Attributes
Point position (vec3)@P
Point normal (vec3)@N
Point color RGB (vec3)@Cd
Point ID number (int)@id
Point number (int)@ptnum
Total point count (int)@numpt
Current time (float)@Time
Current frame (float)@Frame
VEX Code Examples
Add noise to Y position@P.y += noise(@P*2)*0.5;
Color by height@Cd = set(@P.y, 0, 0);
Create custom attributef@myattr = 1.0;
Read slider from nodech("parm_name")
HScript Expressions (Parameters)
Current frame number$F
Current time in seconds$T
Read another node's parameterch("../node/parm")
TipUse i@ prefix for integer attributes, f@ for float, v@ for vector, s@ for string in VEX wrangles. Attribute type prefixes prevent confusion and improve performance.

Procedural VFX Pipeline — Step by Step

InfoThis workflow covers a typical Houdini VFX shot from geometry build to final render composite.
  • 1
    Geometry at OBJ Level: Create a Geometry node at object level. Double-click to enter SOP level. Start with simple primitives (box SOP) and build procedurally: every operation is a new SOP node in sequence.
  • 2
    Build Geometry Network: Chain SOPs: box → scatter → copy_to_points. Use Tab to search and add nodes. Connect outputs to inputs by dragging. The Display flag (eye icon) shows the current result of any node you click.
  • 3
    Add Attributes with VEX: Insert an Attribute Wrangle SOP. Write VEX to modify point positions, colors, or custom attributes. Example: f@pscale = rand(@ptnum) * 2; gives each point a random scale value for Copy to Points.
  • 4
    Set Up Simulation (DOP Network): Create a DOP Network node. Inside, set up solvers (Pyro, FLIP, Vellum) referencing your SOP geometry as the initial state. Adjust substeps and collision geometry.
  • 5
    Cache to Disk: Essential step. Add a File Cache SOP referencing your DOP output. Set an output path ($HIP/cache/sim.$F4.bgeo.sc). Click Save to Disk to cook and cache. Now you can scrub the timeline without recooking.
  • 6
    Lighting & Materials (Karma/Mantra): Create a Karma XPU or Mantra ROP in the out/ context. Set up MaterialX materials for Karma or mantra:surface shaders for Mantra. Place lights in the /obj level (area, HDRI dome).
  • 7
    Render & Composite: Set render frame range in the ROP. Output to EXR image sequence with multiple render passes (beauty, diffuse, specular, shadow, depth). Import passes into Nuke or After Effects for final composite.
